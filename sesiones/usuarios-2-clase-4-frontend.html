Sesión 4 — Frontend Base: HTML, Bootstrap y Plantillas Jinja2

Objetivo

Crear la estructura visual base (plantilla base.html) y las rutas para ingresar y consultar que aún muestren contenido estático.

Paso a Paso

Creación de Carpetas y Ficheros

Estructura del proyecto:

tu_proyecto/
│
├─ app.py
│
├─ templates/
│   ├─ base.html
│   ├─ index.html
│   ├─ ingresar.html
│   └─ consultar.html
│
└─ static/
    ├─ css/
    └─ js/

Fichero base.html

Insertar en el fichero base.html dentro de la carpeta template:

<!doctype html>
<html lang="es">
<head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>{% block title %}StockApp{% endblock %}</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
    <nav class="navbar navbar-expand-lg navbar-light bg-light">
        <div class="container">
            <a class="navbar-brand" href="/">StockApp</a>
            <div class="collapse navbar-collapse">
                <ul class="navbar-nav me-auto">
                    <li class="nav-item"><a class="nav-link" href="/ingresar">Ingresar Stock</a></li>
                    <li class="nav-item"><a class="nav-link" href="/consultar">Consultar Stock</a></li>
                </ul>
            </div>
        </div>
    </nav>
    <main class="container mt-4">
        {% block content %}{% endblock %}
    </main>
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Fichero index.html

Insertar en el fichero index.html dentro de la carpeta template:

{% extends 'base.html' %}
{% block title %}Inicio{% endblock %}
{% block content %}
<div class="text-center">
    <h1 class="mb-4">Bienvenido al Sistema de Gestión de Stock</h1>
    <p>Selecciona una opción en el menú para comenzar.</p>
</div>
{% endblock %}

Fichero consultar.html

Insertar en el fichero consultar.html dentro de la carpeta template:

{% extends 'base.html' %}
{% block title %}Consultar Stock{% endblock %}
{% block content %}
<h2>Consultar productos existentes</h2>
<p>Más adelante mostraremos una tabla con los datos de la base.</p>
{% endblock %}

Fichero ingresar.html

Insertar en el fichero ingresar.html dentro de la carpeta template:

{% extends 'base.html' %}
{% block title %}Ingresar Stock{% endblock %}
{% block content %}
<h2>Ingresar nuevo producto</h2>
<p>En esta vista luego pondremos el formulario.</p>
{% endblock %}

Rutas en Flask

Modificamos el fichero app.py para renderizar las páginas:

from flask import render_template

@app.route('/')
def home():
    return render_template('index.html')

@app.route('/ingresar')
def ingresar_get():
    return render_template('ingresar.html')

@app.route('/consultar')
def consultar_get():
    return render_template('consultar.html')

Ejecutar el Proyecto

En la consola de VSC ejecutamos el siguiente comando dependiendo del S.O.:

# en Windows
venv\Scriptsctivate

# en Linux/Mac
source venv/bin/activate

# Luego ejecutamos
python app.py

Entramos en el navegador a ver las siguientes rutas:

  • http://127.0.0.1:5000/ → Página principal
  • http://127.0.0.1:5000/ingresar → “Ingresar producto”
  • http://127.0.0.1:5000/consultar → “Consultar productos”
