Sesión 3 – Backend con Flask: conexión a MySQL y endpoints básicos

Objetivo: Conectar Flask con MySQL y obtener datos.

Teoría:

  • Estructura básica de una app Flask.
  • Rutas (@app.route).
  • Conexión a base de datos con mysql.connector.

Práctica:

  • Crear app.py.
  • Configurar conexión con stock_db.
  • Ruta inicial / que muestre "Servidor funcionando".
  • Crear ruta /productos que devuelva los productos en formato JSON.
  • Crear ruta /productos/ID que devuelva los productos en formato JSON.
  • Crear ruta /productos/SKU que devuelva los productos en formato JSON.
  • Controlar el filtrado de datos (prevenir que lleguen al Front-end datos no deseados) EJ: password.

Paso a paso

1. Conexión reutilizable (db.py)

Implementar pool de conexiones:

Creamos un fichero db.py y descartamos test_db.py. 

# db.py

import os
from dotenv import load_dotenv
import mysql.connector
from mysql.connector import pooling

load_dotenv()

dbconfig = {
    "host": os.getenv("DB_HOST"),
    "user": os.getenv("DB_USER"),
    "password": os.getenv("DB_PASS"),
    "database": os.getenv("DB_NAME")
}

pool = mysql.connector.pooling.MySQLConnectionPool(pool_name="mypool", pool_size=5, **dbconfig)

def get_connection():
    return pool.get_connection()

2. Endpoints en app.py

Reemplazamos el código antiguo por uno nuevo, creando nuevas rutas (endpoints) para acceder a distintos datos. 

from flask import Flask, jsonify
from db import get_connection

app = Flask(__name__)

@app.route('/')
# Ruta de prueba y principal
def home():
    return 'Servidor Flask funcionando jeje'

@app.route('/productos')
def productos():
    conn = get_connection()
    cursor = conn.cursor(dictionary=True)
    
    # Obtener todos los productos
    cursor.execute("SELECT * FROM productos")
    items = cursor.fetchall()
    
    cursor.close()
    conn.close()
    
    return jsonify(items)

@app.route('/productos/<int:id>')
def producto_por_id(id):
    conn = get_connection()
    cursor = conn.cursor(dictionary=True)
    
    # Buscar producto por ID
    cursor.execute("SELECT * FROM productos WHERE id = %s", (id,))
    item = cursor.fetchone()
    
    cursor.close()
    conn.close()
    
    if item is None:
        return jsonify({"error": "Producto no encontrado"}), 404
    
    return jsonify(item)

@app.route('/productos/sku/<sku>')
def producto_por_sku(sku):
    conn = get_connection()
    cursor = conn.cursor(dictionary=True)
    
    # Buscar producto por SKU + filtrado de campos específicos
    cursor.execute("SELECT id, nombre, descripcion, precio, sku, categoria, cantidad FROM productos WHERE sku = %s", (sku,))
    item = cursor.fetchone()
    
    cursor.close()
    conn.close()
    
    if item is None:
        return jsonify({"error": "Producto no encontrado"}), 404
    
    # Devolver solo los campos que queremos mostrar
    producto_filtrado = {
        "id": item["id"],
        "nombre": item["nombre"],
        "descripcion": item["descripcion"],
        "precio": item["precio"],
        "sku": item["sku"],
        "categoria": item["categoria"],
        "cantidad": item["cantidad"]
    }
    
    return jsonify(producto_filtrado)

if __name__ == '__main__':
    app.run(debug=True)
