Sesión 4: Bucles While

1. Introducción a los Bucles While

El bucle while ejecuta un bloque de código repetidamente mientras una condición sea verdadera (True). Es útil cuando no sabemos exactamente cuántas veces necesitamos repetir algo, pero sí sabemos cuándo debemos parar.

1.1 Estructura básica del while

Ejemplo 1 - Contador simple:

contador = 0

while contador < 5:
    print(f"Contador: {contador}")
    contador += 1  # Esto es igual a: contador = contador + 1

print("Bucle terminado")
⚠ IMPORTANTE: Siempre actualiza la variable de control para evitar bucles infinitos
# Salida:
# Contador: 0
# Contador: 1
# Contador: 2
# Contador: 3
# Contador: 4

Ejemplo 2 - Suma acumulada:

numero = 1
suma_total = 0

while numero <= 10:
    suma_total += numero
    print(f"Sumando {numero}, total acumulado: {suma_total}")
    numero += 1

print(f"La suma de números del 1 al 10 es: {suma_total}")

Ejemplo 3 - Validación de entrada:

# Simular validación de contraseña
contraseña_correcta = "python123"
intentos = 0
max_intentos = 3

while intentos < max_intentos:
    # En la práctica, usarías input() para pedirle al usuario
    contraseña_ingresada = "python12"  # Simular entrada incorrecta
    intentos += 1

    if contraseña_ingresada == contraseña_correcta:
        print("¡Contraseña correcta! Acceso concedido.")
        break  # Salir del bucle
    else:
        print(f"Contraseña incorrecta. Intento {intentos} de {max_intentos}")

if intentos == max_intentos:
    print("Máximo de intentos alcanzado. Acceso denegado.")

2. Bucles While con Listas

2.1 Recorriendo listas con while

Ejemplo 1 - Recorrer lista de nombres:

nombres = ["Ana", "Juan", "María", "Carlos", "Sofía"]
indice = 0

print("Lista de estudiantes:")
while indice < len(nombres):
    print(f"{indice + 1}. {nombres[indice]}")
    indice += 1

print(f"Total de estudiantes procesados: {indice}")

Ejemplo 2 - Procesamiento de calificaciones:

calificaciones = [8.5, 6.0, 9.5, 7.0, 5.5, 10.0, 8.0]
estudiantes = ["Pedro", "Lucía", "Miguel", "Ana", "Carlos", "Sofía", "Diego"]

indice = 0
aprobados = 0
suma_total = 0

print("Reporte de calificaciones:")
while indice < len(calificaciones):
    estudiante = estudiantes[indice]
    calificacion = calificaciones[indice]
    suma_total += calificacion

    if calificacion >= 7.0:
        estado = "APROBADO"
        aprobados += 1
    else:
        estado = "REPROBADO"

    print(f"{estudiante}: {calificacion} - {estado}")
    indice += 1

promedio = suma_total / len(calificaciones)
print(f"\nResumen:")
print(f"Promedio general: {promedio:.2f}")
print(f"Estudiantes aprobados: {aprobados}/{len(calificaciones)}")

Ejemplo 3 - Búsqueda en lista:

productos = ["Laptop", "Mouse", "Teclado", "Monitor", "Bocinas", "Tablet"]
precios = [899.99, 25.50, 75.00, 199.99, 89.99, 299.99]
producto_buscado = "Monitor"

indice = 0
encontrado = False

print(f"Buscando producto: {producto_buscado}")
while indice < len(productos) and not encontrado:
    if productos[indice] == producto_buscado:
        print(f"¡Producto encontrado!")
        print(f"Nombre: {productos[indice]}")
        print(f"Precio: ${precios[indice]}")
        print(f"Posición en inventario: {indice + 1}")
        encontrado = True
    else:
        indice += 1

if not encontrado:
    print(f"Producto '{producto_buscado}' no encontrado en el inventario")

3. Control de Flujo en While

3.1 Break y Continue

Ejemplo 1 - Uso de break:

numeros = [2, 4, 7, 8, 10, 12, 15, 18]
indice = 0

print("Buscando el primer número impar:")
while indice < len(numeros):
    numero_actual = numeros[indice]
    print(f"Verificando: {numero_actual}")

    if numero_actual % 2 != 0:  # Si es impar
        print(f"¡Encontrado! {numero_actual} es impar")
        break  # Salir del bucle inmediatamente

    indice += 1

print("Búsqueda completada")

Ejemplo 2 - Uso de continue:

temperaturas = [18, 25, 32, 15, 28, 35, 12, 30]
indice = 0
dias_calor = 0

print("Analizando temperaturas (solo días con más de 25°C):")
while indice < len(temperaturas):
    temp_actual = temperaturas[indice]

    if temp_actual <= 25:
        indice += 1
        continue  # Saltar al siguiente ciclo

    # Este código solo se ejecuta si temp_actual > 25
    dias_calor += 1
    print(f"Día {indice + 1}: {temp_actual}°C - ¡Día caluroso!")
    indice += 1

print(f"Total de días calurosos (>25°C): {dias_calor}")

Ejemplo 3 - Sistema de menú interactivo:

# Simular un sistema de menú que se repite hasta que el usuario elija salir
menu_activo = True
opcion_usuario = 0  # Simular entrada del usuario
operaciones_realizadas = []

while menu_activo:
    print("\n=== CALCULADORA SIMPLE ===")
    print("1. Sumar")
    print("2. Restar")
    print("3. Multiplicar")
    print("4. Ver historial")
    print("5. Salir")

    # Simular diferentes opciones del usuario
    opciones_simuladas = [1, 2, 3, 4, 5]

    if len(operaciones_realizadas) < len(opciones_simuladas):
        opcion_usuario = opciones_simuladas[len(operaciones_realizadas)]
    else:
        opcion_usuario = 5

    print(f"Opción seleccionada: {opcion_usuario}")

    if opcion_usuario == 5:
        print("¡Gracias por usar la calculadora!")
        break
    elif opcion_usuario == 1:
        resultado = 10 + 5
        operaciones_realizadas.append(f"10 + 5 = {resultado}")
        print(f"Resultado: {resultado}")
    elif opcion_usuario == 2:
        resultado = 15 - 7
        operaciones_realizadas.append(f"15 - 7 = {resultado}")
        print(f"Resultado: {resultado}")
    elif opcion_usuario == 3:
        resultado = 6 * 4
        operaciones_realizadas.append(f"6 * 4 = {resultado}")
        print(f"Resultado: {resultado}")
    elif opcion_usuario == 4:
        print("Historial de operaciones:")
        if operaciones_realizadas:
            for operacion in operaciones_realizadas:
                print(f"- {operacion}")
        else:
            print("No hay operaciones en el historial")
    else:
        print("Opción no válida, intenta de nuevo")
        continue

print("Programa terminado")

4. Aplicaciones Prácticas del While

Ejemplo 1 - Simulador de cajero automático:

saldo_cuenta = 1000.00
operaciones_realizadas = 0
max_operaciones = 3  # Límite diario simulado

print("=== CAJERO AUTOMÁTICO ===")
print(f"Saldo inicial: ${saldo_cuenta}")

# Simular diferentes operaciones
operaciones_simuladas = [
    {"tipo": "consulta"},
    {"tipo": "retiro", "monto": 200},
    {"tipo": "deposito", "monto": 150}
]

indice_operacion = 0

while operaciones_realizadas < max_operaciones and indice_operacion < len(operaciones_simuladas):
    operacion = operaciones_simuladas[indice_operacion]
    print(f"\n--- Operación {operaciones_realizadas + 1} ---")

    if operacion["tipo"] == "consulta":
        print(f"Consultando saldo...")
        print(f"Su saldo actual es: ${saldo_cuenta}")
    elif operacion["tipo"] == "retiro":
        monto = operacion["monto"]
        if monto <= saldo_cuenta:
            saldo_cuenta -= monto
            print(f"Retiro exitoso: ${monto}")
            print(f"Saldo restante: ${saldo_cuenta}")
        else:
            print(f"Fondos insuficientes. Saldo disponible: ${saldo_cuenta}")
    elif operacion["tipo"] == "deposito":
        monto = operacion["monto"]
        saldo_cuenta += monto
        print(f"Depósito exitoso: ${monto}")
        print(f"Nuevo saldo: ${saldo_cuenta}")

    operaciones_realizadas += 1
    indice_operacion += 1

print(f"\nSesión terminada. Operaciones realizadas: {operaciones_realizadas}")

Ejemplo 2 - Juego de adivinanza:

import random

numero_secreto = 42  # En un juego real sería: random.randint(1, 100)
intentos_maximos = 5
intentos_realizados = 0
adivinado = False

# Simular intentos del jugador
intentos_simulados = [30, 50, 45, 42]  # El último intento es correcto

print("=== JUEGO DE ADIVINANZA ===")
print("Adivina el número entre 1 y 100")
print(f"Tienes {intentos_maximos} intentos")

while intentos_realizados < intentos_maximos and not adivinado:
    # Simular entrada del usuario
    if intentos_realizados < len(intentos_simulados):
        intento = intentos_simulados[intentos_realizados]
    else:
        intento = numero_secreto  # Asegurar que termine

    intentos_realizados += 1
    print(f"\nIntento {intentos_realizados}: {intento}")

    if intento == numero_secreto:
        print("¡FELICIDADES! ¡Adivinaste el número!")
        adivinado = True
    elif intento < numero_secreto:
        print("El número es mayor")
    else:
        print("El número es menor")

    if not adivinado:
        intentos_restantes = intentos_maximos - intentos_realizados
        if intentos_restantes > 0:
            print(f"Te quedan {intentos_restantes} intentos")

if not adivinado:
    print(f"\n¡Se acabaron los intentos! El número era: {numero_secreto}")

print(f"Juego terminado. Intentos usados: {intentos_realizados}")

Ejemplo 3 - Procesador de inventario:

productos = ["Laptop", "Mouse", "Teclado", "Monitor", "Bocinas"]
stock = [3, 15, 8, 2, 10]
precios = [899.99, 25.50, 75.00, 199.99, 89.99]

# Procesar productos hasta encontrar uno con stock bajo
indice = 0
productos_revisados = []
stock_bajo_encontrado = False
limite_stock_bajo = 5

print("=== REVISIÓN DE INVENTARIO ===")
print(f"Buscando productos con stock menor a {limite_stock_bajo} unidades\n")

while indice < len(productos) and not stock_bajo_encontrado:
    producto_actual = productos[indice]
    stock_actual = stock[indice]
    precio_actual = precios[indice]

    productos_revisados.append(producto_actual)

    print(f"Revisando: {producto_actual}")
    print(f"Stock actual: {stock_actual} unidades")
    print(f"Precio: ${precio_actual}")

    if stock_actual < limite_stock_bajo:
        print("⚠ ¡ALERTA! Stock bajo detectado")
        print("Se requiere reabastecimiento urgente")
        stock_bajo_encontrado = True

        # Calcular sugerencia de reabastecimiento
        cantidad_sugerida = 20 - stock_actual
        costo_reabastecimiento = cantidad_sugerida * precio_actual
        print(f"Cantidad sugerida a comprar: {cantidad_sugerida} unidades")
        print(f"Costo estimado: ${costo_reabastecimiento:.2f}")
    else:
        print("✅ Stock suficiente")

    print("-" * 40)
    indice += 1

print(f"\nResumen de revisión:")
print(f"Productos revisados: {len(productos_revisados)}")
print(f"Lista de productos revisados: {productos_revisados}")

if stock_bajo_encontrado:
    print("⚠ Se requiere acción inmediata para reabastecimiento")
else:
    print("✅ Todos los productos revisados tienen stock adecuado")
