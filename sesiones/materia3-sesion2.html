1. Conversión de Tipos

Python permite convertir un tipo de dato a otro usando funciones como int(), float(), str() y bool().

# De cadena (str) a entero (int)
edad = "25"
edad_entero = int(edad)
print(edad_entero) # Output: 25 (int)
print(type(edad_entero)) # Output: <class 'int'>

La función incorporada print(): su propósito es mostrar información en la salida estándar (generalmente la consola).

La función incorporada type(): Se usa para obtener el tipo de dato de un objeto (int, boolean, string, float). Esto ayuda a identificar el tipo del objeto que se encuentra guardado en la variable (edad_entero).

# De entero (int) a cadena (str)
puntaje = 100
puntaje_str = str(puntaje)
print("Tu puntaje es: " + puntaje_str) # Output: "Tu puntaje es: 100"
# De flotante (float) a entero (int)
precio = 99.99
precio_entero = int(precio) # Trunca los decimales
print(precio_entero) # Output: 99 (int)

b) Conversiones con input()

# Entrada de usuario (siempre es str)
anio_nacimiento = input("Ingresa tu año de nacimiento: ") # Ejemplo: "2005"
anio_actual = 2024

La función incorporada input(): Su propósito es capturar entradas del usuario desde la consola y devolverlas como una cadena de texto (str).

# Convertir a int para operar
edad = anio_actual - int(anio_nacimiento)
print(f"Tienes {edad} años") # Output: "Tienes 19 años"

La f luego de print se llama f-string y permite formatear cadenas de texto. Se usa para insertar variables dentro de cadenas de texto de forma más legible. Por ejemplo, en lugar de concatenar "Hola, " + nombre + "!", se puede usar f"Hola, {nombre}!". Esto hace que el código sea más limpio y fácil de entender.
nombre = "Ana"
edad = 25
print(f"Hola, {nombre}. Tienes {edad} años.")
# Output: Hola, Ana. Tienes 25 años.

2. Operadores de Comparación

Comparan dos valores y devuelven True o False.

  • Operadores: ==, !=, >, <, >=, <=.
a) Comparaciones numéricas
a = 10
b = 5
print(a == b) # False
print(a != b) # True
print(a > b) # True
print(a <= 10) # True

b) Comparaciones con cadenas (str)
# Comparación lexicográfica (orden alfabético)
print("Python" == "python") # False (diferencia mayúsculas/minúsculas)
print("apple" < "banana") # True ("a" viene antes que "b")

c) Comparaciones mixtas
# Entero vs Flotante
print(5 == 5.0) # True (Python ignora el tipo en comparaciones numéricas)
print(10 == "10") # False (int vs str)

  • ¿Por qué "10" == 10 en Python devuelve False? ¿Cómo harías para compararlos correctamente?

• "10" es una cadena y 10 es un entero.
• Para compararlos, convierte uno de los tipos: int("10") == 10 → True.
