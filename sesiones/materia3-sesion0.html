Objetivo

Introducir al estudiante en la importancia de la documentación técnica y los pasos fundamentales para la instalación de entornos de desarrollo y herramientas necesarias para la materia.

1. ¿Por qué es importante la documentación?

  • Facilita la comprensión y el mantenimiento de proyectos.
  • Permite la colaboración efectiva entre desarrolladores.
  • Reduce errores y acelera la resolución de problemas.
  • Ejemplo: Manuales de usuario, guías de instalación, comentarios en el código.
Documentación Tutorial (lectura)

2. Instalación de Herramientas Básicas

  • Editor de código: Visual Studio Code, Sublime Text.
  • Control de versiones: Git y GitHub.
  • Lenguajes y entornos: Node.js, Python, Java (según la materia).
  • Terminal: Uso básico de la terminal de comandos.
Instalar Python Descargar VS Code

3. Buenas Prácticas de Instalación

  • Leer siempre la documentación oficial antes de instalar.
  • Verificar requisitos del sistema.
  • Preferir instaladores oficiales y evitar fuentes desconocidas.
  • Ejecutar pruebas básicas tras la instalación para confirmar el funcionamiento.

4. Material de apoyo

  • Paginas web y videos en youtube ofrecen gran variedad de contenido para aprender
Curso Python (youtube) Curso Python (lectura y test)

Conclusión

La documentación y la correcta instalación de herramientas son la base para un desarrollo eficiente y seguro. Dominar estos aspectos iniciales facilita el aprendizaje y la resolución de problemas a lo largo del curso.
