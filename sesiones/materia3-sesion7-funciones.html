Sesión 7: Funciones

¿Qué es una función?

  • Bloque de código reutilizable.
  • Evita repetir código (principio DRY: Don't Repeat Yourself).
  • Mejora la organización y legibilidad.

Sintaxis básica de def

def nombre_funcion(parametros):
    # código a ejecutar
    return valor

Declaramos que será una función al escribir "def" luego le damos un nombre a la función y abrimos paréntesis (), dentro de los paréntesis () puede ir vacío o con parámetros dentro.

Escribimos la lógica / código que nos interese. Usamos return si queremos que algo dentro del código / lógica que escribimos se "imprima", veremos en consola como si hubiéramos escrito print(valor).

Para que el código de una función se ejecute necesitamos llamar a la función, esto se hace escribiendo el nombre de la función creada más paréntesis.

Ejemplo básico:

def saludar():
    print("Hola, humano!")

saludar()

Mini-ejercicio:

  • Crear una función que imprima tu nombre y ciudad.

Parámetros y argumentos

Teoría:

  • Parámetros → variables definidas en la función.
  • Argumentos → valores reales que pasamos.
  • Tipos de parámetros:
    • Posicionales
    • Con valor por defecto
    • Keyword arguments

Ejemplos:

# Parámetro posicional
def saludar_persona(nombre): # parámetro = nombre
    print(f"Hola, {nombre}!")

saludar_persona("Ana") # argumento = "Ana"

# Parámetro con valor por defecto
def saludar_persona(nombre="Mundo"):
    print(f"Hola, {nombre}!")

saludar_persona()

Valores de retorno (return)

Teoría:

  • return envía un resultado al lugar donde se llamó la función.
  • Si no hay return, la función devuelve None.

Ejemplo:

def suma(valor_uno, valor_dos): # parámetros
    return valor_uno + valor_dos

resultado = suma(5, 3) # argumentos

print(resultado) # 8

En este ejemplo se ingresan dos parámetros a la función suma: valor_uno y valor_dos.

Posteriormente se llama a la función y dentro de los paréntesis se ingresaron 2 valores (argumentos), cada valor se guarda en un parámetro valor_uno, valor_dos respetando el orden de posición, tener mucho cuidado al introducir argumentos en la posición incorrecta, esto puede dar resultados no deseables.

El argumento 5 se guarda en valor_uno y 3 en valor_dos, se suman y return devuelve "8", este número 8 se guarda en la variable resultado y posteriormente se imprime.

Ejercicio rápido:

  • Escribir una función que reciba dos números y devuelva su promedio.

Alcance de variables (Scope)

Teoría:

  • Variables locales: existen solo dentro de la función.
  • Variables globales: existen en todo el programa.
  • No abusar de globales.

Ejemplo:

x = 10
def mostrar():
    y = 5
    print(x) # Accede a variable global
    print(y) # Variable local

mostrar()

# print(y) # Error: no existe fuera de la función
