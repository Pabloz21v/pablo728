Sesión 10: Sets y Diccionarios

Parte 1: Sets (Conjuntos)

Teoría

  • ¿Qué es un set? Colección no ordenada, mutable y de elementos únicos (sin duplicados).
  • Usos comunes:
    • Eliminar duplicados de una lista.
    • Operaciones de conjuntos (unión, intersección, diferencia).
    • Verificar pertenencia de forma eficiente.

Sintaxis Básica

# Crear un set
mi_set = {1, 2, 3}
otro_set = set([4, 5, 6]) # Desde una lista
# Como pueden ver un set se declara usando llaves {}
# Para que recuerden una lista usa corchetes [] y una tupla paréntesis ()

Operaciones Principales

Operación Sintaxis Ejemplo
Añadir elemento set.add(elemento) mi_set.add(4)
Eliminar set.remove(elemento) mi_set.remove(2)
Unión set1.union(set2) ó set1 | set2 {1, 2} {3, 4} → {1, 2, 3, 4}
Intersección set1 & set2 {1, 2} & {2, 3} → {2}
Diferencia set1 - set2 {1, 2} - {2} → {1}

Ejemplos Prácticos

# Eliminar duplicados de una lista
numeros = [1, 2, 2, 3, 4, 4]
sin_duplicados = set(numeros)
print(sin_duplicados) # Output: {1, 2, 3, 4}
# Convierte una lista a un set, el cual elimina los valores repetidos.

# Verificar pertenencia
vocales = {"a", "e", "i", "o", "u"}
print("a" in vocales) # Output: True

Intersection / Intersección

Cuando usamos este método intersection() o el signo ampersand &, los valores dentro de los sets que persistirán en el nuevo set creado serán los valores que existen en los sets involucrados. 

# Unimos set1 y set2 con intersection():
set1 = {"apple", "banana", "cherry"}
set2 = {"google", "microsoft", "apple"}

set3 = set1.intersection(set2)
# ó
set3 = set1 & set2
print(set3) # "apple"
# En el nuevo set solo veremos el valor "apple"

Difference / Diferencia

El método difference() o el signo menos - devuelve los valores dentro del primer set que no se repitan en el segundo. 

set1 = {"apple", "banana", "cherry"}
set2 = {"google", "microsoft", "apple"}

set3 = set1.difference(set2)
# ó
set3 = set1 - set2

print(set3) # {'banana', 'cherry'}

Symmetric Difference / Diferencia Simétrica

El método symmetric_difference() o el símbolo careta ^ devuelve los valores de los sets que no se repiten en ambos. 

set1 = {"apple", "banana", "cherry"}
set2 = {"google", "microsoft", "apple"}

set3 = set1.symmetric_difference(set2)
# ó
set3 = set1 ^ set2

print(set3) # {'google', 'banana', 'microsoft', 'cherry'}

Parte 2: Diccionarios

Teoría

  • ¿Qué es un diccionario? Colección de pares clave-valor. Las claves son únicas e inmutables.
  • Usos comunes:
    • Almacenar datos estructurados (ej: información de usuarios).
    • Contadores (ej: frecuencia de palabras).

Sintaxis Básica

# Crear un diccionario
estudiante = {
    "nombre": "Ana",
    "edad": 16,
    "curso": "Python"
}
# Se declara el diccionario, en este caso lo llamamos “estudiante” usamos el signo igual “=” como siempre para declarar que se guardará algo en esta variable “estudiante” y abrimos llave, a diferencia del set lo que irá dentro es la lógica de clave-valor. En este ejemplo las claves son “nombre”, “edad” y “curso”, y los valores respectivos a continuación de los dos puntos “:”.

Operaciones Principales

Operación Sintaxis Ejemplo
Acceder a valor dic["clave"] estudiante["nombre"] → "Ana"
Añadir/Modificar dic["clave"] = valor estudiante["edad"] = 17
Eliminar del dic["clave"] del estudiante["curso"]
Obtener claves dic.keys() list(estudiante.keys())
Obtener valores dic.values() list(estudiante.values())

Ejemplos Prácticos

# Contador de palabras
texto = "hola hola mundo"
palabras = texto.split() # ['hola', 'hola', 'mundo']
contador = {} # diccionario declarado y vacío

for palabra in palabras:
    contador[palabra] = contador.get(palabra, 0) + 1

print(contador) # Output: {"hola": 2, "mundo": 1}
# El método Split() separa una oración y la guarda en una lista separándolas por una coma “,”.
# En la primera iteración el bucle for palabra = hola esta sería una de las claves que se guardan dentro del diccionario contador y el valor como 0, luego suma 1 al valor.
# Segunda iteración, palabra = hola, como ya existe esta clave no se creará de nuevo dentro del diccionario, pero sí se suma 1 al valor, ahora siendo 2.
# Tercer iteración palabra = mundo, como no existe en el dicc. crea esta clave = mundo y su valor = 0, y luego suma 1 a valor, ahora siendo 1. # Output: {"hola": 2, "mundo": 1}

# Diccionarios anidados
usuarios = {
    "ana123": {"edad": 16, "email": "ana@example.com"},
    "luis456": {"edad": 17, "email": "luis@example.com"}
}

print(usuarios["ana123"]["edad"]) # Output: 16
# Dentro de un diccionario se anida otro diccionario.
# Primero podemos ver el diccionario principal “usuarios” y su clave – valor.
# Las claves del diccionario usuarios son: ana123 y luis345, pero a su vez estos se vuelven los diccionarios de otras clave-valor
# ana123 es el diccionario de {"edad": 16, "email": "ana@example.com"}
# Y luis456 es el diccionario de: {"edad": 17, "email": "luis@example.com"}
# Si queremos acceder al valor del email de luis456:
print(usuarios["luis456"]["email"]) # Output: luis@example.com

Recorrer un diccionario anidado:

myfamily = {
    "hijo1": {
        "name": "Emil",
        "year": 2004
    },
    "hijo2": {
        "name": "Tobias",
        "year": 2007
    },
    "hijo3": {
        "name": "Linus",
        "year": 2011
    }
}

for clave, valor in myfamily.items():
    print(clave)
    for clave_del_dicc_anidado in valor:
        print(clave_del_dicc_anidado + ':', valor[clave_del_dicc_anidado])
# Como pueden ver en este ejemplo, existe el diccionario “myfamily” el cual contiene 3 diccionarios “hijo1”, “hijo2” y “hijo3”.
# Para recorrer todos los valores de cada clave se usa un bucle for dentro de otro for.
# El primer bucle for recorre el diccionario “myfamily”, guardando en la variable "clave" = hijo1 y en "valor" = {"name": "Emil", "year": 2004} para la primera iteración, hijo2 para la segunda iteración e hijo3 para la tercera iteración respectivamente.
# Esto se puede realizar gracias al método items() el cual devuelve la clave:valor como tupla “()” dentro de una lista “[]”:
# Ej: dict_items([(hijo1, {'name': 'Emil', 'year': 2004}), (hijo2, {'name': 'Tobias', 'year': 2007}), (hijo3, {'name': 'Linus', 'year': 2011})])
# Vamos a analizar la primera iteración "clave" = hijo1 y "valor" = {"name": "Emil", "year": 2004}
# Como podrán ver se guardó la clave y el valor en cada variable.
# Ahora pasamos al segundo bucle for. Utilizamos los datos guardados dentro de valor "valor" = {"name": "Emil", "year": 2004}. Para la primera iteración guardará en "clave_del_dicc_anidado" solo la clave, en este caso "name".
# Luego pasamos al print() aquí se imprimirá clave_del_dicc_anidado = name ":", valor(clave_del_dicc_anidado) = el valor correspondiente a "name" es decir, Emil.
# Para la segunda iteración del for será clave_del_dicc_anidado = year ":", valor(clave_del_dicc_anidado) = el valor correspondiente a "year" es decir, 2004.
# Los bucles for anidados son algo difícil de comprender, si ingresan el código pueden visualizar mejor lo que ocurre en cada recorrido.
