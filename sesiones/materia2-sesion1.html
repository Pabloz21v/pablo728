Sesión 1: "Tipos de Archivos, Sistemas de Almacenamiento y Comparativas"

1. Tipos de Archivos Comunes

Los archivos son como "cajas digitales" que guardan información. Cada tipo tiene una extensión (ej: .docx) que indica su formato y cómo abrirlo.

Tipo de Archivo Extensión ¿Para qué sirve? Ejemplo de Uso Características Únicas
Documento de texto .txt, .docx, .odt Guardar textos simples o con formato Tareas escolares, informes. Editable, compatible con procesadores de texto.
Documento electrónico .pdf Compartir información con diseño fijo (texto, imágenes, gráficos). Trabajos finales, manuales, formularios. Mantiene el formato en cualquier dispositivo, permite contraseñas y firmas digitales.
Hoja de cálculo .xlsx, .ods Organizar datos en tablas y cálculos Presupuestos, listas de notas. Funciones matemáticas y gráficos dinámicos.
Presentación .pptx, .odp Crear diapositivas para exposiciones Proyectos de clase, charlas. Animaciones, transiciones y elementos multimedia.
Imagen .jpg, .png, .gif Almacenar fotos o gráficos Memes, fotos de redes sociales. .gif soporta animaciones.
Audio .mp3, .wav Guardar música o sonidos Playlist de Spotify, podcasts. .wav es de alta calidad (sin compresión).
Video .mp4, .avi Reproducir películas o clips TikToks, videos de YouTube. .mp4 es ligero y compatible.
Ejecutable .exe, .apk Instalar programas Instalador de Minecraft, Zoom. Solo funciona en su sistema operativo designado. .apk aplicaciones para celulares.
Programación .html, .js, .php, .java Crear páginas web o aplicaciones Páginas web (HTML), apps móviles (Java). Requiere software especializado para editar.
Archivos Comprimidos .zip, .rar Reducir el tamaño de archivos o agrupar varios en uno Enviar múltiples archivos por correo, ahorrar espacio. .rar permite contraseñas y división en partes.

De programación:

  • HTML (.html): Estructura páginas web (ej: el esqueleto de un sitio).
  • JavaScript (.js): Agrega interactividad a una web (ej: botones que cambian de color).
  • PHP (.php): Maneja datos en servidores (ej: formularios de login).
  • Java (.java): Desarrolla aplicaciones multiplataforma (ej: apps de Android).

2. Comparación: .odt (Abierto) vs. .docx (Propietario)

Aspecto .odt (OpenDocument) .docx (Microsoft Word)
Creador Comunidad de código abierto Microsoft (empresa privada)
Software LibreOffice, Google Docs Microsoft Word, Office 365
Ventajas
  • Gratuito.
  • Compatible con múltiples programas.
  • Menos riesgo de obsolescencia.
  • Formato estándar en empresas.
  • Funciones avanzadas (ej: diseño gráfico).
Desventajas
  • Menos herramientas avanzadas.
  • Puede haber errores de formato al abrir en Word.
  • Requiere licencia de pago.
  • Dependencia de Microsoft.

Ejemplo: Si guardas un trabajo en .odt y lo abres en Word, algunos elementos (tablas, fuentes) podrían verse distintos. Por eso, si usas Word, es mejor guardar en .docx.

Esto ocurre debido a que algunas extensiones se pueden llegar a abrir con otros programas de funcionalidad similar, pero no garantizan que se pueda aprovechar todas las características del programa original. Como también intentar abrirlo con programas similares no garantiza que funcionen.

Para poder cambiar la extensión de un archivo, en la carpeta contenedora ir a la pestaña "vista", en ella del lado derecho encontrarán "extensiones de nombre de archivo" tildar esta opción, esto permite ver la extensión a la derecha del nombre del archivo, seguido por un punto y luego la extensión.

Saber cómo darle una extensión a un archivo es importante debido a que en ocasiones al descargar un archivo o por problemas de guardado este no agrega su extensión, dando a interpretar que este se a dañado, solo hay que escribir su extensión nuevamente para tenerlo funcional.

Ejercicio: el profesor les administra un archivo .rar con varios archivos sin extensión, descomprimirlo en una carpeta, cada archivo otorga pistas de cuál es su posible extensión, darle la extensión y ejecutarlo, si logran acceder y visualizar su contenido es que se ha realizado correctamente.
