Sesión 5: Listas

Objetivo Profundizado: Dominar operaciones usando listas.

1. Listas

Definición: Colecciones ordenadas y mutables (pueden modificarse). Permite valores duplicados y mezcla de tipos de datos.

Se inicia abriendo corchetes [] y dentro del mismo agregar valores separados por el uso de una coma “,”. Cada valor tiene una posición (índice) partiendo de la posición 0.

Operaciones Básicas

Acción Sintaxis Ejemplo
Crear lista lista = [elementos] numeros = [1, 2, 3]
Acceder por índice lista[índice] numeros[0] → 1
Modificar elemento lista[índice] = nuevo_valor numeros[1] = 20 → [1, 20, 3]
Añadir elemento lista.append(elemento) numeros.append(4) → [1, 20, 3, 4]
Eliminar elemento lista.remove(elemento) numeros.remove(20) → [1, 3, 4]
Longitud len(lista) len(numeros) → 3

Ejemplos

# Crear lista de nombres
nombres = ["Ana", "Luis", "Carlos"]

print(nombres[1]) # Output: "Luis"

# Añadir elementos
nombres.append("María")

print(nombres) # Output: ["Ana", "Luis", "Carlos", "María"]

# Eliminar elementos
nombres.remove("Luis")

print(nombres) # Output: ["Ana", "Carlos", "María"]

Verificar si el valor existe

Para determinar si un valor específico existe en la lista se usa la palabra clave in: 

# Verifica si "apple" existe en la lista:
canasta_de_frutas = ["apple", "banana", "cherry"]
if "apple" in canasta_de_frutas:
    print("Yes, 'apple' is in the fruits list")

Insertar valores

Para insertar un nuevo valor, sin reemplazar otros valores existentes, podemos usar el método insert() para insertarlo en una posición específica. 

# Insertar "orange" en la posición/index 1:
canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_de_frutas.insert(1, "orange")
print(canasta_de_frutas) # Output: ['apple', 'orange', 'banana', 'cherry']

Extender Listas

Para unir listas usar el método extend() (permite añadir tuplas, sets, diccionarios). 

# Agregar canasta_tropical a canasta_de_frutas:
canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_tropical = ["mango", "pineapple", "papaya"]
canasta_de_frutas.extend(canasta_tropical)
print(canasta_de_frutas) # Output: ['apple', 'banana', 'cherry', 'mango', 'pineapple', 'papaya']

# Ejemplo: Agregar una tupla a una lista:
canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_de_frutas_tupla = ("kiwi", "orange")
canasta_de_frutas.extend(canasta_de_frutas_tupla)
print(canasta_de_frutas) # Output: ['apple', 'banana', 'cherry', 'kiwi', 'orange']

Eliminar un valor en una posición (índice) específica

El método pop() elimina un valor en una posición específica. 

# Remove the second item: Elimina el segundo valor (índice = 1)
canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_de_frutas.pop(1)
print(canasta_de_frutas) # Output: ['apple', 'cherry']

# Si no se especifica la posición, borra el último elemento

Limpiar la lista

Vacía la lista con el método clear().

canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_de_frutas.clear()
print(canasta_de_frutas) # Output: []

Bucles en Listas

Se puede recorrer los valores de una lista usando el método for: 

# El recorrido y el uso de print() posteriormente imprime los valores que va recorriendo, uno a uno.
canasta_de_frutas = ["apple", "banana", "cherry"]
for fruta in canasta_de_frutas:
    print(fruta) # Output: apple banana cherry

Recorriendo por número de índice

Se puede recorrer por la cantidad de índices usando el método range() y len() para crear una iteración controlable. 

# Imprime todos los valores basado en el rango determinado.
canasta_de_frutas = ["apple", "banana", "cherry"]
for fruta in range(len(canasta_de_frutas)):
    print(canasta_de_frutas[fruta]) # Output: apple banana cherry

Usando el loop While

Usando el método len() determinamos la longitud de la lista. Nunca olvidar agregar una variable que se vaya incrementando en 1 unidad por cada iteración, esto permite cortar el bucle una vez llegue al valor de la longitud de la lista. 

# Imprime todos los valores usando while, basándose en la posición (índice)
canasta_de_frutas = ["apple", "banana", "cherry"]
posicion_index = 0
while posicion_index < len(canasta_de_frutas):
    print(canasta_de_frutas[posicion_index])
    posicion_index = posicion_index + 1 # Output: apple banana cherry

Recorriendo usando Compresión de listas / List Comprehension

La comprensión de listas ofrece una sintaxis más corta para recorrer las listas: 

# La forma acortada del bucle for imprimirá todos los ítems de la lista:
canasta_de_frutas = ["apple", "banana", "cherry"]

for fruta in canasta_de_frutas:
    print(fruta) # Output: apple banana cherry

Crear nuevas listas a partir de una anterior y aplicando un filtro, en este caso la letra “a”, recorre la lista canasta_de_frutas, verifica en cada posición del índice su valor, a ese valor analiza si contiene la letra “a” y de ser así, esta se guarda en la nueva lista canasta_de_frutas_con_letra_a. 

canasta_de_frutas = ["apple", "banana", "cherry", "kiwi", "mango"]
canasta_de_frutas_con_letra_a = []

for fruta in canasta_de_frutas:
    if "a" in fruta:
        canasta_de_frutas_con_letra_a.append(fruta)

print(canasta_de_frutas_con_letra_a) # Output: ['apple', 'banana', 'mango']

Una forma aún más comprimida:

# La lectura del código [fruta for fruta in canasta_de_frutas if "a" in fruta] se realizaría de la siguiente forma:
# La primera variable “fruta” es donde se guarda la lógica final, el resto se lee así: por cada (for) fruta en (in) canasta_de_frutas si (if) la letra “a” se encuentra en el valor dentro de la variable fruta, usar la variable fruta (hace referencia a la primera variable fruta) y guardar su valor en la nueva lista canasta_de_frutas_con_letra_a.

canasta_de_frutas = ["apple", "banana", "cherry", "kiwi", "mango"]

canasta_de_frutas_con_letra_a = [fruta for fruta in canasta_de_frutas if "a" in fruta]

print(canasta_de_frutas_con_letra_a) # Output: ['apple', 'banana', 'mango']

La sintaxis:

lista_nueva = [expression for item in iterable if condition == True]

Cambia los valores en una nueva lista, pasándolos a mayúscula.

canasta_de_frutas = ["apple", "banana", "cherry", "kiwi", "mango"]

nuevo_canasta = [fruta.upper() for fruta in canasta_de_frutas]

print(nuevo_canasta) # Output: ['APPLE', 'BANANA', 'CHERRY', 'KIWI', 'MANGO']

upper() convierte todo a mayúscula.

lower() convierte todo a minúscula.

sort() ordena alfabéticamente y ascendente (números).

sort(reverse = True) para que ordene en sentido opuesto y descendente. 

# Otro ejemplo con sort(reverse = True)
canasta_de_frutas = ["orange", "mango", "kiwi", "pineapple", "banana"]

canasta_de_frutas.sort(reverse = True)

canasta_de_frutas = [fruta.upper() for fruta in canasta_de_frutas]

print(canasta_de_frutas) # Output: ['PINEAPPLE', 'ORANGE', 'MANGO', 'KIWI', 'BANANA']

Usar el método copy()

Para crear una copia de una lista usamos el método copy(), list(), [:]. Esto permite que la nueva copia ocupe un espacio en memoria totalmente distinto al de la original. 

canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_nueva = canasta_de_frutas.copy()
print(canasta_nueva)

# o

canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_nueva = list(canasta_de_frutas)
print(canasta_nueva)

# o

canasta_de_frutas = ["apple", "banana", "cherry"]
canasta_nueva = canasta_de_frutas[:]
print(canasta_nueva)

Unir 2 listas

Se puede sumar una lista con la otra “+”:

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list3 = list1 + list2
print(list3) # Output: ['a', 'b', 'c', 1, 2, 3]

O usar el método append() recorriendo los valores de la segunda lista y agregando uno a uno a la primera lista: 

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

for x in list2:
    list1.append(x)

print(list1)

O usar el método extend() para agregar una lista al final de la primera: 

list1 = ["a", "b", "c"]
list2 = [1, 2, 3]

list1.extend(list2)
print(list1)

Métodos para las listas

Método Descripción
append() Agregar un elemento/valor al final de la lista.
clear() Vacía la lista.
copy() Copia la lista y la nueva tendrá otro espacio en memoria.
extend() Agregar los elementos de una lista/tupla/etc al final de la lista actual.
insert() Agrega un elemento a una posición específica.
pop() Remueve un elemento en una posición específica.
remove() Remueve un valor específico en la lista.
reverse() Invierte el orden de la lista.
sort() Ordena la lista.
