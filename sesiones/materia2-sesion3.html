Sesión 3: Recuperación de Datos

Objetivo: Aprender técnicas y herramientas para recuperar archivos eliminados o dañados, y comprender las limitaciones y buenas prácticas para prevenir la pérdida de datos.

Contenido Teórico

1. Principios de la Recuperación de Datos

¿Cómo funciona?

  • Los archivos eliminados no se borran inmediatamente; el sistema marca el espacio como disponible.
  • Herramientas escanean el disco en busca de estos datos "invisibles".

Diferencias HDD vs SSD

  • En HDD, los datos persisten hasta ser sobrescritos.
  • En SSD, el comando TRIM prepara celdas en memoria para ser sobrescritas.

Tipos de pérdida de datos

  • Eliminación accidental: Recuperación desde la Papelera de Reciclaje o con software.
  • Daño lógico: Corrupción de archivos por virus o errores del sistema.
  • Daño físico: Fallos en discos duros o USBs (requiere profesionales).

2. Límites y Precauciones

  • Archivos sobrescritos: Si se guardan nuevos datos, la recuperación es imposible ya que se está sobrescribiendo sobre el archivo previamente borrado.
  • Cuanto antes se traten de recuperar los datos, aumentan las chances de lograrlo.
  • Tratar de instalar en otro disco (HDD o SSD) un programa para recuperar datos para evitar sobrescribir datos producto de la instalación del programa de recuperación.

3. Herramientas Modernas de Recuperación

Software gratuito

  • Disk Drill (PC):
    1. Descargar Disk-Drill
  • Recuperar Fotos – All Recovery (Celular desde la app store, autor AnviGame).

Servicios en la nube

  • Restauración de versiones anteriores (Google Drive (Vault): Es un servicio pago de Google Drive para recuperar datos borrados).

Actividades Prácticas

Actividad 1: "Rescate de Archivos Borrados"

  • Objetivo: Recuperar archivos eliminados usando Disk Drill y Recuperar Fotos – All Recovery.
  • Pasos:
    1. USBs y/o celulares con archivos borrados (documentos, fotos).
    2. Los estudiantes usan Disk Drill (PC) o Recuperar Fotos – All Recovery (Móvil) para escanear el dispositivo.
    3. Clasifican los archivos recuperables por tipo y prioridad (ej: fotos vs documentos).
    4. Observar el estado de los archivos recuperados.
    5. ¿Qué factores afectan la recuperación (tiempo, uso del dispositivo)?

  • Herramientas: Windows Backup, Time Machine (macOS).
