TP 2° MySQL Parte uno

En grupos de entre 2-4 integrantes deberán realizar las siguientes actividades. Se dedicarán 2 clases para dicho trabajo.

Ir guardando los códigos que vayan creando y usando en un archivo Word, ya que será el material que me enviarán para testearlo posteriormente y/o evaluarlos.

  • La entrega del TP se realiza por EMAIL.
  • Debe incluir los nombres y apellidos de los integrantes.

Actividades por Tema

1. Creación y Gestión de Bases de Datos y Tablas

Objetivo: Evaluar el uso de CREATE DATABASE, DROP DATABASE, USE, CREATE TABLE IF NOT EXISTS, DROP TABLE, y tipos de datos.

  • Si el grupo es de 2 integrantes crearán 1 tabla.
  • Si son 3 o 4 crearán 2 tablas.
  • Las tablas pueden ser de la temática que más les guste.
  1. TABLA: dicha tabla debe contener como mínimo 6 columnas aplicando los tipos de datos aprendidos.
    • Primera columna llamada id (no olvidar asignarla como clave primaria y auto incremental, teoría de la sesión 8).
    • VARCHAR
    • INT
    • DOUBLE
    • BOOLEAN
    • TEXT
    • Usar NOT NULL en alguna/s columna/s (que tenga sentido por qué no debe estar vacía dicha o dichas columnas).
  2. Aplicar “IF NOT EXISTS” al crear tablas o DDBB.
  3. Incluir código para eliminar tanto las tablas como las DDBB.
  4. No olvidar usar “USE” para posicionarse en la DDBB donde trabajen.

2. Inserción y Consulta de Datos

Objetivo: Evaluar el uso de INSERT INTO, SELECT * FROM, y LIMIT.

  1. Insertar 4 registros por tabla (ósea 4 filas “VALUES”).
  2. Insertar 2 registros más por tabla repitiendo algunos datos de los 4 primeros registros y dejando algunos espacios vacíos (precaución con dejar valores vacíos en la columna/s que designaron con NOT NULL).
  3. Mostrar los primeros 3 registros.

3. Actualización y Eliminación de Datos

Objetivo: Evaluar el uso de UPDATE, SET, WHERE, DELETE, y TRUNCATE.

  1. Actualizar el valor de algún registro basado en la columna con tipo de dato VARCHAR.
  2. Actualizar el valor de algún registro basado en la columna con tipo de dato INT.
  3. Eliminar una fila/registro basado en su id.
  4. Vaciar la tabla.
  5. Volver a insertar los registros a la tabla.

4. Consultas con Condiciones

Objetivo: Evaluar el uso de WHERE, AND, OR, NOT, >=, <=, <>, =, >, <, IS NULL, IS NOT NULL.

  1. Crear un filtro usando AND.
  2. Crear un filtro usando OR.
  3. Crear un filtro usando NOT.
  4. Crear un filtro usando alguno de los 3 “>=, <=, <>”.
  5. Crear un filtro usando alguno de los 3 “=, >, <,”.
  6. Crear un filtro usando IS NULL o IS NOT NULL.

5. Ordenamiento y Valores Distintos

Objetivo: Evaluar el uso de ORDER BY (ASC/DESC) y DISTINCT.

  1. Ordenar los registros de mayor a menor por la columna que contenga el tipo de dato VARCHAR.
  2. Ordenar los registros de menor a mayor por la columna que contenga el tipo de dato DOUBLE.
  3. Mostrar los registros basado en la columna con tipo de dato INT.
