Sesión 9: Arquitectura Avanzada y Componentes Especializados

BIOS/UEFI y CMOS

BIOS (Basic Input/Output System)

El BIOS es el firmware que se ejecuta cuando se enciende la computadora, responsable de inicializar el hardware y cargar el sistema operativo.

Funciones del BIOS:

  • POST (Power-On Self Test): Verifica el funcionamiento del hardware.
  • Inicialización del hardware: Configura componentes básicos.
  • Bootloader: Localiza y carga el sistema operativo.
  • Interfaz de configuración: Setup para modificar parámetros del sistema.
  • Servicios de tiempo real: Funciones básicas de entrada/salida.

Limitaciones del BIOS tradicional:

  • Modo real de 16 bits: Acceso limitado a memoria.
  • Interfaz de texto: Solo teclado, sin mouse.
  • Tamaño de disco: Limitado a 2.2 TB (MBR).
  • Velocidad de arranque: Proceso más lento.
  • Seguridad limitada: Vulnerable a ataques de rootkit.

UEFI (Unified Extensible Firmware Interface)

Sucesor moderno del BIOS, introducido para superar las limitaciones del firmware tradicional.

Ventajas del UEFI:

  • Interfaz gráfica: Soporte para mouse y elementos visuales.
  • Arranque rápido: Secure Boot y Fast Boot.
  • Soporte GPT: Discos de más de 2.2 TB.
  • Arquitectura modular: Drivers y aplicaciones UEFI.
  • Seguridad mejorada: Secure Boot contra malware.
  • Networking: Capacidad de red desde el firmware.

Funciones avanzadas del UEFI:

  • Overclocking: ajuste de voltaje, frecuencia de CPU/RAM.
  • XMP/EXPO: perfiles para memoria RAM de alto rendimiento.
  • Monitorización: temperaturas, voltajes, RPM de ventiladores.
  • Boot Priority: orden de arranque (SSD, USB, red).
  • Secure Boot: bloquea sistemas no firmados (útil para Windows 11).
  • Fast Boot: omite chequeos para arranque rápido.

CMOS (Complementary Metal-Oxide Semiconductor)

Memoria no volátil que almacena la configuración del BIOS/UEFI y mantiene el reloj en tiempo real.

Características del CMOS:

  • Alimentación: Batería de litio CR2032 (3V).
  • Consumo: Muy bajo, batería dura 3-5 años.
  • Contenido: Configuración de hardware, fecha, hora.
  • Tamaño: Típicamente 256 bytes.
  • Acceso: Solo a través del setup del BIOS/UEFI.

Problemas comunes relacionados con CMOS:

  • Batería agotada: Pérdida de configuración y hora.
  • Configuración corrupta: Fallos de arranque.
  • Clear CMOS: Procedimiento para resetear configuración.

Northbridge y Southbridge: Evolución de la Arquitectura

Arquitectura Tradicional (1990s-2000s)

El chipset tradicional se dividía en dos chips principales que manejaban diferentes funciones del sistema.

Northbridge (MCH - Memory Controller Hub):

  • Funciones principales:
    • Controlador de memoria RAM.
    • Interfaz con el procesador (FSB).
    • Slot AGP para tarjeta gráfica.
    • Comunicación con el Southbridge.
  • Características:
    • Chip de alta velocidad y temperatura.
    • Requería disipador de calor.
    • Determinaba el tipo de RAM soportada.
    • Bus del sistema (FSB) pasaba por aquí.

Southbridge (ICH - I/O Controller Hub):

  • Funciones principales:
    • Controlador de dispositivos de E/S.
    • Puertos USB, SATA, IDE.
    • Slots PCI.
    • Audio integrado.
    • Controlador de red.
    • BIOS/UEFI.
  • Características:
    • Menor velocidad que Northbridge.
    • Mayor cantidad de funciones integradas.
    • Comunicación con Northbridge vía bus dedicado.

Arquitectura Moderna (2008-Actualidad)

La integración de funciones en el procesador cambió radicalmente la arquitectura del chipset.

Integración en el CPU:

  • Controlador de memoria: Integrado en el procesador desde 2003 (AMD) y 2008 (Intel).
  • Controlador PCIe: Lanes PCIe principales manejadas por el CPU.
  • Controlador gráfico: GPU integrada en muchos procesadores.
  • QPI/HyperTransport: Reemplazo del FSB tradicional.

Chipset Moderno (PCH - Platform Controller Hub):

El chipset moderno es esencialmente un Southbridge mejorado que se comunica directamente con el CPU.

  • Funciones actuales:
    • Gestiona PCIe lanes adicionales (para tarjetas secundarias, NVMe, Wi-Fi).
    • Controladores SATA y USB.
    • Controla red Ethernet, Wi-Fi/Bluetooth integrados.
    • Audio de alta definición (HD).
    • Administra puertos USB (cantidad, versión 2.0/3.2/4.0).
    • Soporte para overclocking.

Socket de CPU: Tipos y Compatibilidad

Lugar donde se instala el microprocesador. Debe coincidir exactamente con el modelo de CPU.

Tipos de Socket

PGA (Pin Grid Array):

  • Diseño: Pines en el procesador, agujeros en el socket.
  • Ventajas: Socket más barato de fabricar.
  • Desventajas: Pines frágiles en el CPU.
  • Ejemplos: Socket AM4, AM5 (AMD).

LGA (Land Grid Array):

  • Diseño: Pines en el socket, contactos planos en el CPU.
  • Ventajas: CPU más resistente, mejor transferencia térmica.
  • Desventajas: Socket más costoso y frágil.
  • Ejemplos: LGA1700, LGA1200 (Intel).

BGA (Ball Grid Array):

  • Diseño: CPU soldado permanentemente.
  • Uso: Laptops, dispositivos móviles.
  • Ventajas: Compacto, eficiente térmicamente.
  • Desventajas: No es removible ni actualizable.

Consideraciones por Factor de Forma

ATX (Advanced Technology eXtended):

  • Ventajas: Máxima expansión, mejor refrigeración, más conectores.
  • Desventajas: Requiere gabinete grande, mayor costo.
  • Ideal para: Gaming, workstations, sistemas con múltiples tarjetas.

Micro-ATX:

  • Ventajas: Buen equilibrio tamaño/características, precio moderado.
  • Desventajas: Limitaciones de expansión.
  • Ideal para: Oficina, gaming básico, sistemas de propósito general.

Mini-ITX:

  • Ventajas: Muy compacto, bajo consumo, silencioso.
  • Desventajas: Expansión muy limitada, mayor costo por características.
  • Ideal para: HTPC, sistemas portátiles, espacios reducidos.

La elección del factor de forma afecta:

  1. Cantidad de slots PCIe y RAM.
  2. Número de puertos SATA/USB.
  3. Compatibilidad con gabinetes y fuentes de poder.

Componentes Adicionales y Buffers

Componentes de Soporte

Capacitores:

  • Función: Filtrado y estabilización de voltaje.
  • Tipos: Electrolíticos, cerámicos, tantalio.
  • Problemas: Capacitores hinchados indican fallas.
  • Calidad: Capacitores japoneses considerados superiores.

Reguladores de Voltaje (VRM):

  • Función: Conversión de 12V a voltajes específicos del CPU.
  • Fases: Más fases = mejor estabilidad y eficiencia.
  • Componentes: MOSFETs, inductores, capacitores.
  • Importancia: Crítico para overclocking y estabilidad.

Buffers y Controladores

Buffer de datos:

  • Función: Amplificación y estabilización de señales.
  • Ubicación: Entre chipset y slots de expansión.
  • Importancia: Previene interferencias y pérdida de señal.

Controladores de Bus:

  • USB Controllers: Manejo de múltiples puertos USB.
  • SATA Controllers: Gestión de dispositivos SATA.
  • Network Controllers: Conectividad Ethernet/Wi-Fi.
  • Audio Codecs: Procesamiento de audio integrado.

Tecnologías de Protección

  • ESD Protection: Protección contra descargas electrostáticas.
  • Over-voltage Protection: Protección contra sobrevoltajes.
  • Thermal Protection: Monitoreo y protección térmica.
  • Short Circuit Protection: Protección contra cortocircuitos.

La calidad de los componentes de soporte (VRM, capacitores, reguladores) es tan importante como las características principales de la placa madre. Una VRM deficiente puede limitar el rendimiento del procesador y la estabilidad del sistema, especialmente en aplicaciones exigentes o overclocking.
