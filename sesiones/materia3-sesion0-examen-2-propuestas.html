Propuestas de Proyectos en Python

Las siguientes propuestas son opciones de miniproyectos que pueden elegir para el próximo examen
Pueden elegir estas opciones o inventar las suyas, la idea es que apliquen los conocimientos alcanzados hasta el momento
  • Condicionales (if), Match
  • Funciones
  • Listas, Sets, Tuplas, Diccionarios
  • Bucles while, for
  • Operadores y tipos de datos
El examen será como el anterior, solo que en esta ocasión elegiré al azar cual miniproyecto defenderá cada uno
En grupo de hasta 4 personas deben hacer 1 miniproyecto por integrante, dedicaremos 4 clases aprox para su desarrollo

1. Juegos

a) Ahorcado con categorías

  • Descripción: El usuario elige una categoría (ej: animales, países) y adivina la palabra.
  • Conceptos: while, for, match, listas, sets, funciones.

b) Piedra, papel o tijera (versión extendida)

  • Descripción: El usuario juega contra la computadora con opciones adicionales (ej: lagarto, Spock).
  • Conceptos: while, match, listas, funciones.

c) Trivia con niveles de dificultad

  • Descripción: Preguntas con diferentes puntuaciones según la dificultad.
  • Conceptos: for, match, diccionarios, funciones.

2. Herramientas de Productividad

a) Gestor de tareas con prioridades

  • Descripción: Permite agregar tareas con prioridad (alta, media, baja) y filtrarlas.
  • Conceptos: while, listas, diccionarios, funciones.

b) Gestor de contraseñas

  • Descripción: Almacena contraseñas de forma segura (simulada) y permite buscarlas.
  • Conceptos: while, diccionarios, funciones.

c) Lista de compras con presupuesto

  • Descripción: Permite agregar productos y calcula el total gastado.
  • Conceptos: for, listas, funciones.

3. Programas Educativos

a) Calculadora matemáticas (mejorar la que hicieron)

  • Descripción: Genera ejercicios aleatorios de suma, resta, multiplicación y división.
  • Conceptos: while, match, funciones.

b) Aprendizaje de tablas de multiplicar

  • Descripción: El usuario practica una tabla específica y recibe retroalimentación.
  • Conceptos: for, listas, funciones.

c) Juego de capitales del mundo

  • Descripción: El usuario adivina la capital de un país seleccionado al azar.
  • Conceptos: while, diccionarios, funciones.

d) Simulador de examen

  • Descripción: Genera un examen con preguntas aleatorias y calcula la nota final.
  • Conceptos: for, listas, diccionarios, funciones.

4. Análisis de Datos Simples

a) Analizador de textos

  • Descripción: Cuenta palabras, letras y frecuencia de palabras en un texto.
  • Conceptos: for, diccionarios, sets, funciones.

b) Calculadora de estadísticas

  • Descripción: Calcula media, mediana y moda de una lista de números.
  • Conceptos: for, listas, funciones.

c) Filtro de números pares/impares

  • Descripción: Filtra números pares e impares de una lista.
  • Conceptos: for, listas, funciones.

d) Contador de vocales

  • Descripción: Cuenta cuántas vocales hay en una frase.
  • Conceptos: for, strings, funciones.

5. Automatización

a) Generador de informes automáticos

  • Descripción: Crea un informe en formato .txt con datos de una lista.
  • Conceptos: for, listas, funciones.
  • Nota: Les tocará investigar cómo crear un fichero .txt desde el código y guardar los datos en el mismo.

c) Enviar correos masivos (simulado)

  • Descripción: Simula el envío de correos a una lista de destinatarios.
  • Conceptos: for, listas, diccionarios.

6. Simulaciones

a) Simulador de lotería

  • Descripción: Genera números aleatorios y verifica si el usuario ganó.
  • Conceptos: for, sets, funciones.

7. Programas de Utilidad

a) Conversor de monedas

  • Descripción: Convierte entre diferentes monedas usando tasas de cambio.
  • Conceptos: match, diccionarios, funciones.

b) Calculadora de IMC

  • Descripción: Calcula el Índice de Masa Corporal (IMC) de una persona.
  • Conceptos: match, funciones.
