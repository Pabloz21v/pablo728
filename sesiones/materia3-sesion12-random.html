Ejemplos de uso del método random() en Python

Este método les permite generar un numero aleatorio entre 0 y 1

También les presento usos para que puedan aplicar en sus proyectos

No olvidar colocar import random arriba de todo el codigo, alguno para la calculadora ya lo hicieron con import math

1. Generar un número aleatorio entre 0 y 1


import random
numero_aleatorio = random.random()
print(numero_aleatorio)  # Ejemplo de salida: 0.5488135039273248

2. Simular el lanzamiento de una moneda


import random
resultado = "Cara" if random.random() < 0.5 else "Cruz"
print(resultado)  # Ejemplo de salida: "Cara" o "Cruz"

3. Generar un número aleatorio en un rango específico


import random
minimo = 5
maximo = 10
numero_aleatorio = minimo + (maximo - minimo) * random.random()
print(numero_aleatorio)  # Ejemplo de salida: 7.3456789

4. Seleccionar un elemento aleatorio de una lista


import random
colores = ["rojo", "verde", "azul", "amarillo"]
color_aleatorio = colores[int(random.random() * len(colores))]
print(color_aleatorio)  # Ejemplo de salida: "azul"

5. Simular una probabilidad (ejemplo: 30% de éxito)


import random
probabilidad_exito = 0.3
if random.random() < probabilidad_exito:
    print("¡Éxito!")
else:
    print("Fracaso.")

6. Generar un retraso aleatorio en un programa


import random
import time
tiempo_espera = 1 + 4 * random.random()  # Entre 1 y 5 segundos
time.sleep(tiempo_espera)
print("Espera terminada.")

7. Simular un dado de 6 caras


import random
resultado_dado = int(random.random() * 6) + 1
print(resultado_dado)  # Ejemplo de salida: 3
