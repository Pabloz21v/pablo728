2. For con √çndices - enumerate()

Cuando necesitamos tanto el elemento como su posici√≥n, usamos enumerate(). Es m√°s elegante que manejar un contador manual.

Ejemplo 1 - Lista numerada:

tareas = ["Estudiar Python", "Hacer ejercicio", "Leer libro", "Llamar a familia"]

print("=== MI LISTA DE TAREAS ===")
for indice, tarea in enumerate(tareas):
    print(f"{indice + 1}. {tarea}")

print("\n=== VERSI√ìN ALTERNATIVA ===")
for numero, tarea in enumerate(tareas, start=1):
    print(f"{numero}. {tarea}")

Ejemplo 2 - Ranking de puntuaciones:

jugadores = ["Alice", "Bob", "Carol", "David", "Eva"]
puntuaciones = [950, 1200, 800, 1100, 1050]

print("=== RANKING DE JUGADORES ===")
for posicion, (jugador, puntos) in enumerate(zip(jugadores, puntuaciones)):
    medalla = ""
    if posicion == 0:
        medalla = "ü•á"
    elif posicion == 1:
        medalla = "ü•à"
    elif posicion == 2:
        medalla = "ü•â"
    print(f"{posicion + 1}¬∞. {jugador}: {puntos} puntos {medalla}")

Ejemplo 3 - Modificaci√≥n controlada de listas:

productos = ["laptop", "mouse", "teclado", "monitor"]
precios = [899.99, 25.50, 75.00, 199.99]

descuento = 0.10

print("=== APLICANDO DESCUENTOS ===")
for indice, (producto, precio) in enumerate(zip(productos, precios)):
    if precio > 100:
        precio_con_descuento = precio * (1 - descuento)
        print(f"{indice + 1}. {producto.title()}: ${precio:.2f} ‚Üí ${precio_con_descuento:.2f} ‚≠ê")
        productos_modificados.append(f"{producto} (con descuento)")
    else:
        print(f"{indice + 1}. {producto.title()}: ${precio:.2f} (sin cambio)")
        productos_modificados.append(producto)

print(f"\nProductos procesados: {len(productos_modificados)}")
print(f"Productos con descuento: {len([p for p in productos_modificados if 'descuento' in p])}")

3. Control de Flujo en For

3.1 Break y Continue en For

4. For Anidados y Aplicaciones Avanzadas

4.1 Bucles for anidados

Ejemplo 1 - Tabla de multiplicar completa:

print("=== TABLAS DE MULTIPLICAR (1-5) ===")

    print(f"\n--- Tabla del {tabla} ---")
    for multiplicador in range(1, 11):
        resultado = tabla * multiplicador
        print(f"{tabla} x {multiplicador} = {resultado}")
print("\n¬°Todas las tablas completadas!")

Ejemplo 2 - An√°lisis de ventas por tienda:

    [1200, 1500, 1800, 1600, 2000, 2200, 1400],
    [1000, 1200, 1400, 1300, 1700, 1900, 1100],
    [1500, 1800, 2000, 1900, 2300, 2500, 1600]
]
dias_semana = ["Lun", "Mar", "Mi√©", "Jue", "Vie", "S√°b", "Dom"]

print("=== REPORTE DE VENTAS SEMANALES ===")
for indice_tienda, tienda in enumerate(tiendas):
    print(f"\n{tienda}:")
    ventas_tienda = ventas_semanales[indice_tienda]
    total_tienda = 0

        venta_dia = ventas_tienda[indice_dia]
        total_tienda += venta_dia
        print(f" {dia}: ${venta_dia}")

    print(f" Total semanal: ${total_tienda}")
    print(f" Promedio diario: ${promedio_tienda:.2f}")

Ejemplo 3 - Sistema de calificaciones por materia:

    [7.0, 8.0, 6.5],
    [8.5, 9.0, 10.0]

print("=== REPORTE ACAD√âMICO COMPLETO ===")
for indice_estudiante, estudiante in enumerate(estudiantes):
    print(f"\nüìö Estudiante: {estudiante}")
    califs_estudiante = calificaciones[indice_estudiante]
    suma_califs = 0
    for indice_materia, materia in enumerate(materias):
        calificacion = califs_estudiante[indice_materia]
        suma_califs += calificacion

            estado = "‚≠ê EXCELENTE"
        elif calificacion >= 7.0:
            estado = "‚úÖ APROBADO"
        else:
            estado = "‚ö† NECESITA MEJORAR"
        print(f" {materia}: {calificacion} - {estado}")
    promedio_estudiante = suma_califs / len(materias)
    print(f" üìä Promedio general: {promedio_estudiante:.2f}")

print(f"\n=== PROMEDIOS POR MATERIA ===")
for indice_materia, materia in enumerate(materias):
    suma_materia = 0
    for indice_estudiante in range(len(estudiantes)):
        suma_materia += calificaciones[indice_estudiante][indice_materia]
    promedio_materia = suma_materia / len(estudiantes)
    print(f"{materia}: {promedio_materia:.2f}")

5. Comprensiones de Lista (List Comprehensions)

Las comprensiones de lista son una forma concisa de crear listas usando for en una sola l√≠nea.

Ejemplo 1 - Comparaci√≥n tradicional vs comprensi√≥n:

numeros = [1, 2, 3, 4, 5]
cuadrados_tradicional = []
for numero in numeros:
    cuadrados_tradicional.append(numero ** 2)

cuadrados_comprension = [numero ** 2 for numero in numeros]

print(f"M√©todo tradicional: {cuadrados_tradicional}")
print(f"Con comprensi√≥n: {cuadrados_comprension}")
print(f"¬øSon iguales? {cuadrados_tradicional == cuadrados_comprension}")

Ejemplo 2 - Filtrado con comprensi√≥n:

edades = [15, 22, 17, 30, 16, 25, 19, 40]
nombres = ["Ana", "Juan", "Mar√≠a", "Carlos", "Sof√≠a", "Pedro", "Luc√≠a", "Diego"]

mayores_edad = [nombre for nombre, edad in zip(nombres, edades) if edad >= 18]

print(f"Mayores de edad: {mayores_edad}")
print(f"Total: {len(mayores_edad)} personas")

Ejemplo 3 - Transformaci√≥n de datos:

productos = ["laptop", "mouse", "teclado", "monitor"]
precios = [899.99, 25.50, 75.00, 199.99]

productos_info = [f"{producto.title()}: ${precio}" for producto, precio in zip(productos, precios)]
productos_descuento = [f"{producto.title()} (con descuento)" for producto, precio in zip(productos, precios) if precio > 100]

print("Informaci√≥n de productos:")
for info in productos_info:
    print(f"- {info}")

print(f"\nProductos con descuento: {productos_descuento}")
