Sesión 1: Variables, Tipos de Datos y Función Print

1. Declaración de Variables

¿Qué es una variable?

Una variable es un espacio de memoria que almacena un valor. En Python, las variables se crean automáticamente cuando les asignamos un valor por primera vez.

Formas Correctas de Declarar Variables:

Ejemplo 1:
nombre = "Juan"
edad = 25
altura = 1.75
Ejemplo 2:
mi_nombre = "Ana"
numero_favorito = 7
es_estudiante = True
Ejemplo 3:
precio_producto = 99.99
cantidad_stock = 50
producto_disponible = True

Formas Incorrectas de Declarar Variables:

Ejemplo 1 (Incorrecto):
2nombre = "Juan" # No puede empezar con número
mi-edad = 25 # No se permiten guiones
class = "A" # 'class' es una palabra reservada
Ejemplo 2 (Incorrecto):
mi nombre = "Ana" # No se permiten espacios
$precio = 100 # No se permite el símbolo $
for = 5 # 'for' es una palabra reservada
Ejemplo 3 (Incorrecto):
9altura = 1.80 # No puede empezar con número
mi@email = "..." # No se permite @
if = True # 'if' es una palabra reservada

2. Sensibilidad a Mayúsculas y Minúsculas

Python es case sensitive, lo que significa que distingue entre mayúsculas y minúsculas.

Ejemplo 1:
nombre = "Juan"
NOMBRE = "Carlos"
# Estas son 2 variables diferentes
Ejemplo 2:
edad = 20
EDAD = 30
print(edad) # Imprime: 20
print(EDAD) # Imprime: 30
Ejemplo 3:
miVariable = 100
mivariable = 200
MIVARIABLE = 300
# Son 3 variables completamente distintas

3. Convenciones de Nomenclatura

Convención Descripción Uso en Python
snake_case Palabras separadas por guiones bajos ✅ Recomendado
camelCase Primera palabra en minúscula, siguientes capitalizadas ❌ No recomendado

Ejemplos de snake_case (Recomendado en Python):

Ejemplo 1:
nombre_completo = "Juan Pérez"
fecha_nacimiento = "1990-05-15"
numero_telefono = "123456789"
Ejemplo 2:
precio_total = 150.50
cantidad_productos = 3
descuento_aplicado = 0.10
Ejemplo 3:
es_mayor_edad = True
tiene_licencia = False
ciudad_residencia = "Madrid"

Ejemplos de camelCase (No recomendado en Python):

Ejemplo 1:
nombreCompleto = "Juan Pérez"
fechaNacimiento = "1990-05-15"
numeroTelefono = "123456789"
Ejemplo 2:
precioTotal = 150.50
cantidadProductos = 3
descuentoAplicado = 0.10
Ejemplo 3:
esMayorEdad = True
tieneLicencia = False
ciudadResidencia = "Madrid"

4. La Función print()

La función print() se utiliza para mostrar información en la pantalla.

Ejemplo 1 - Imprimir texto simple:
print("¡Hola, mundo!")
print('Python es genial')
print("Estoy aprendiendo programación")
Ejemplo 2 - Imprimir variables:
nombre = "María"
edad = 28
print(nombre)
print(edad)
print("Mi nombre es", nombre, "y tengo", edad, "años")
Ejemplo 3 - Diferentes formas de imprimir:
producto = "Laptop"
precio = 899.99
print("Producto:", producto)
print(f"El precio de {producto} es ${precio}")
print("Total: $" + str(precio))

5. Tipos de Datos Básicos

5.1 Números Enteros (int)

Ejemplo 1:
edad = 25
año_actual = 2024
temperatura = -5
print(type(edad)) # <class 'int'>
Ejemplo 2:
cantidad_estudiantes = 30
numero_negativo = -100
cero = 0
print("Estudiantes:", cantidad_estudiantes)
Ejemplo 3:
puntos_juego = 1500
nivel = 8
vidas = 3
print(f"Nivel {nivel}, Puntos: {puntos_juego}, Vidas: {vidas}")

5.2 Cadenas de Texto (str)

Ejemplo 1:
nombre = "Ana García"
apellido = 'Rodríguez'
mensaje = "Bienvenido al curso de Python"
print(type(nombre)) # <class 'str'>
Ejemplo 2:
ciudad = "Barcelona"
pais = "España"
codigo_postal = "08001"
print(ciudad, pais, codigo_postal)
Ejemplo 3:
email = "usuario@ejemplo.com"
contraseña = "mi_contraseña_123"
descripcion = "Esta es una descripción larga del producto"
print("Email:", email)

5.3 Números Decimales (float)

Ejemplo 1:
precio = 29.99
altura = 1.75
peso = 68.5
print(type(precio)) # <class 'float'>
Ejemplo 2:
promedio_notas = 8.7
temperatura = 23.5
distancia = 15.25
print(f"Promedio: {promedio_notas}")
Ejemplo 3:
salario = 2500.50
descuento = 0.15
pi = 3.14159
print("Salario:", salario, "Descuento:", descuento)

5.4 Valores Booleanos (bool)

Ejemplo 1:
es_estudiante = True
tiene_trabajo = False
mayor_edad = True
print(type(es_estudiante)) # <class 'bool'>
Ejemplo 2:
llueve = False
hace_sol = True
dia_laborable = True
print("¿Llueve?", llueve)
Ejemplo 3:
juego_terminado = False
usuario_conectado = True
archivo_guardado = True
print(f"Juego terminado: {juego_terminado}")
