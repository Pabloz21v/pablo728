Qué Debemos Saber Hasta Este Punto

Cómo Nombrar las Variables:

  1. Nombres Descriptivos:
    • Usa nombres que describan claramente el propósito o contenido de la variable. Esto hace que el código sea más legible y comprensible.
    • Ejemplo: nombre_usuario en lugar de nu.
  2. Convención de Nombres:
    • snake_case: En Python, es común usar "snake case" para nombrar variables, es decir, palabras en minúsculas separadas por guiones bajos.
    • Ejemplo: precio_producto, cantidad_estudiantes.
  3. Evitar Nombres Genéricos:
    • Evita nombres como lista, dato, valor, etc., a menos que su propósito sea absolutamente claro en el contexto.
  4. No Usar Palabras Reservadas:
    • No uses palabras reservadas de Python como nombres de variables (por ejemplo, if, else, for, while, etc.).
  5. Sensibilidad a Mayúsculas y Minúsculas:
    • Los nombres de variables en Python son sensibles a mayúsculas y minúsculas, por lo que edad y EDAD son variables diferentes.
    • Evitar usar Edad (empezar con una letra mayúscula y continuar con minúscula) esto se usa para las "clases" tema que veremos más adelante.
  6. Constantes:
    • Para constantes, se suele usar nombres en mayúsculas con guiones bajos separando las palabras.
    • Ejemplo: PI = 3.14159, VELOCIDAD_LUZ.
  7. Evitar Caracteres Especiales:
    • No uses caracteres especiales como @, #, $, etc., en los nombres de variables.
  8. Longitud del Nombre:
    • No temas usar nombres largos si son descriptivos. Es mejor tener un nombre largo y descriptivo que uno corto y ambiguo.
  9. Nombres Únicos y Significativos:
    • Asegúrate de que cada variable tenga un nombre único y significativo dentro del ámbito en el que se utiliza.
  10. Evitar Nombres Engañosos:
    • No uses nombres que puedan inducir a error sobre el tipo de dato o el propósito de la variable.

Formas Correctas

myvar = "Pablo"
my_var = "Pablo"
_my_var = "Pablo"
myVar = "Pablo"
MYVAR = "Pablo"
myvar2 = "Pablo"

Formas Incorrectas

2myvar = "Pablo"
my-var = "Pablo"
my var = "Pablo"

Cuando Comparamos:

  • Es igual a: a == b
  • No es igual a: a != b
  • Menor que: a < b
  • Menor o igual a: a <= b
  • Mayor que: a > b
  • Mayor o igual que: a >= b

Incluso dentro de un condicional como IF, ELIF, lo que devuelve esa comparación es un booleano TRUE o FALSE.

En el caso de estar en un condicional, si el resultado de la comparación es TRUE, ingresará a las líneas pertenecientes a ese if / elif; de ser FALSE, continuará con el siguiente if / elif hasta finalizar en else por no haber encontrado un TRUE en todo el recorrido de comparaciones. Recordar que en else no compara, esto ocurre ya que nada de lo anterior fue TRUE.

Uso de las Triples Comillas:

  • Las triples comillas permiten escribir con saltos de línea, esto facilita y reduce la cantidad de print() que podríamos usar.

Ejemplo con Comillas Simples:

    print("hola")
    print("espero que")
    print("sus vacaciones")
    print("fueran buenas")

Ejemplo con Comillas Triples:

    print("""hola
        espero que
        sus vacaciones
        fueran buenas""")
