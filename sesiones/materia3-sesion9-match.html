Sesión 9: Match

La sentencia Match sirve como otra opción de if-elif-else.

1. Introducción

¿Qué es Match?

  • Permite devolver un resultado al encontrar una coincidencia.

¿Por qué usar match?

  • Código más legible y conciso.
  • Manejo de estructuras complejas (listas, diccionarios, clases).

Diferencias con if-elif-else:

  • match permite desestructurar datos y comparar patrones directamente.

2. Sintaxis Básica

match expresion:
    case x:
        code block
    case y:
        code block
    case z:
        code block
    case _:
        code block

x, y, z son el valor que busca coincidir con lo contenido dentro de la variable "expresion".

El valor guardado dentro de "expresion" buscará una coincidencia entre las opciones, al encontrarla ejecuta su bloque de código correspondiente.

Al igual que en if-else, podemos usar un tipo de else usando el guion bajo "_" este actúa como else, es decir, si no se encuentran coincidencias hará lo contenido en su bloque de código.

Ejemplo:

def describir_estacion(estacion):
    match estacion:
        case "primavera":
            return "Flores y colores"
        case "verano":
            return "Calor y playa"
        case "otoño":
            return "Hojas caídas"
        case "invierno":
            return "Frío y nieve"
        case _:
            return "Estación desconocida"

aspecto_estacion = describir_estacion("invierno")
print(aspecto_estacion)  # Salida: Frío y nieve
# Si cambiamos el texto del argumento en la función describir_estacion(), print() imprimirá otro resultado debido a que hará match con otro valor.

3. Más de un valor por case

day = 4
match day:
    case 1 | 2 | 3 | 4 | 5:
        print("Es un día de semana")
    case 6 | 7:
        print("VAMOSSS es findeeee!")
# Podemos dar más de una opción de coincidencia por caso (case), en este ejemplo la variable day guarda un día específico, se declara match y la variable, empieza la búsqueda de coincidencias, en el primer case se indican 5 valores separados por una barra vertical "|", y en el segundo case 2 valores separados por la barra vertical.

4. Agregar if a match:

Ejemplo: Match con listas y guards

def describir_lista_avanzado(lista):
    match lista:
        case []:
            return "Lista vacía"
        case [x] if x > 10:
            return f"Lista con un elemento mayor a 10: {x}"
        case [x, y] if x == y:
            return f"Lista con dos elementos iguales: {x}"
        case [x, *rest] if len(rest) > 0:
            return f"Lista con más de un elemento, primero: {x}"
        case _:
            return "Lista no reconocida"

que_contiene_la_lista = []
conclusion = describir_lista_avanzado(que_contiene_la_lista)
print(conclusion)
# Este ejemplo aplica muchos contenidos, funciones, listas, match, if dentro de match.
# Si agregan y cambian los valores dentro de la lista que se guarda en "que_contiene_la_lista", podrán ver los distintos comportamientos. Intenten ejecutar todos los case.
