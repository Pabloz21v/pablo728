Sesión 10: PLACAS DE VIDEO: FUNDAMENTOS Y ARQUITECTURA DE GPUs

Introducción a las Placas de Video (GPU)

¿Qué es una GPU?

La Unidad de Procesamiento Gráfico (GPU - Graphics Processing Unit) es un procesador especializado diseñado para acelerar la creación y renderizado de imágenes, video y gráficos. A diferencia del CPU que tiene pocos núcleos potentes, la GPU tiene cientos o miles de núcleos más simples diseñados para procesamiento paralelo masivo.

Las GPUs modernas no solo aceleran gráficos, sino también cargas de trabajo de IA (ej. inferencia de modelos como LLaMA o Stable Diffusion), simulaciones científicas (ej. dinámica de fluidos) y procesamiento de datos masivos (ej. big data con CUDA/OpenCL).

Ejemplo: La NVIDIA H200 (2024) está optimizada para IA con 94GB de HBM3e y núcleos Tensor de 4ª generación.

Diferencias entre CPU y GPU

Característica CPU GPU
Núcleos 8-128 núcleos Miles de núcleos
Diseño Latencia baja, ejecución secuencial Throughput masivo, paralelismo extremo
Memoria Cache L3/L4 (hasta 256MB) Memoria dedicada de alta velocidad. VRAM dedicada (GDDR7/HBM3e)
Aplicaciones Tareas generales, single-thread Gráficos, IA, renderizado, simulaciones
Consumo (TDP) 15-250W 70-600W

Throughput masivo: capacidad para procesar grandes volúmenes de datos de manera simultánea y eficiente.

Tipos de Implementación de GPU

GPU Integrada (iGPU):

  • Ubicación: Integrada en el procesador (APU en AMD)
  • Memoria: Comparte RAM del sistema
  • Rendimiento: Básico a moderado
  • Consumo: Muy bajo
  • Aplicaciones: Oficina, multimedia básico, laptops
  • Ejemplos: Intel UHD Graphics, AMD Radeon Graphics

GPU Dedicada (dGPU):

  • Ubicación: Tarjeta de expansión separada
  • Memoria: VRAM dedicada (GDDR6/GDDR7)
  • Rendimiento: Alto a extremo
  • Consumo: Alto (50-600W)
  • Aplicaciones: Gaming, profesional, minería cripto
  • Ejemplos: NVIDIA GeForce, AMD Radeon RX

Arquitectura de GPU Moderna

Componentes Principales

Núcleos de Sombreado (Shader Cores):

  • Función: Procesamiento de píxeles y vértices
  • NVIDIA: CUDA Cores
  • AMD: Stream Processors
  • Cantidad: De cientos a varios miles por GPU

Unidades de Textura (TMUs):

  • Función: Aplicación de texturas a objetos 3D
  • Importancia: Determina la calidad visual
  • Medida: Gigatexels por segundo

Unidades de Salida de Render (ROPs):

  • Función: Escritura final de píxeles al framebuffer
  • Impacto: Rendimiento en altas resoluciones
  • Características: Anti-aliasing, blending

Memoria de Video (VRAM)

Tipos de Memoria de Video

GDDR (Graphics Double Data Rate):

  • GDDR5: Ancho de banda hasta 336 GB/s
  • GDDR6: Ancho de banda hasta 672 GB/s
  • GDDR6X: Ancho de banda hasta 1 TB/s
  • GDDR7: Ancho de banda hasta 1.5 TB/s
  • Características: Alta velocidad, optimizada para gráficos

HBM (High Bandwidth Memory):

  • HBM2: Hasta 1.8 TB/s de ancho de banda
  • HBM3: Hasta 3.2 TB/s de ancho de banda
  • HBM3e: Hasta 5 TB/s de ancho de banda
  • Ventajas: Ancho de banda extremo, menor consumo
  • Desventajas: Costo elevado, complejo de fabricar
  • Uso: GPUs profesionales y de gama alta
Tipo de VRAM Velocidad Ancho de Bus Ancho de Banda Uso Típico
GDDR5 7-8 Gbps 256-bit 336 GB/s Gama media-baja
GDDR6 14-16 Gbps 256-bit 672 GB/s Gama media-alta
GDDR6X 19-21 Gbps 384-bit 1008 GB/s Gama alta
GDDR7 28-32 Gbps 384-bit 1.4-1.7 TB/s Nuevas GPUs 2025 (ej. RTX 5090)
HBM2e 3.6 Gbps 4096-bit 1800 GB/s Profesional
HBM3 6.4 Gbps 5120-bit 3.2 TB/s IA/Data Center (ej. H200)
HBM3e 8.0 Gbps 6144-bit 5.0 TB/s GPUs de próxima gen (2025-2026)

Importancia de la VRAM

  • Texturas: Almacenamiento de texturas de alta resolución
  • Framebuffers: Imágenes renderizadas
  • Shaders: Programas de sombreado
  • Geometría: Datos de modelos 3D
  • Cálculos: Datos temporales de procesamiento

Gaming:

  • 1080p: 6-8GB (ej. RX 7600).
  • 1440p/4K: 12-16GB (ej. RTX 4070 Ti).
  • 8K/VR: 24GB+ (ej. RTX 5090).

IA/ML:

  • Modelos como Stable Diffusion XL requieren 12GB+ de VRAM.
  • GPUs profesionales (ej. NVIDIA H200) tienen 94GB HBM3e para entrenar LLMs.
  • Minería.

BIOS

¿Dónde está el BIOS en una GPU?

  • Ubicación física: Es un chip de memoria flash soldado directamente en el PCB de la GPU.
  • Ejemplo: En una RTX 4090, el chip del BIOS suele estar cerca del conector PCIe, marcado como "BIOS".

¿Qué hace el BIOS de una GPU?

  • Funciones clave:
    • Inicialización: Configura la GPU al encender el sistema (ej: frecuencias base, voltajes, memoria).
    • Compatibilidad: Define cómo la GPU se comunica con el CPU, la placa madre y el sistema operativo.
    • Overclocking y límites: Establece los límites de potencia (TDP), frecuencias máximas y curvas de ventilador.
    • Soporte para tecnologías: Habilita funciones como ray tracing o DLSS (según el modelo).
