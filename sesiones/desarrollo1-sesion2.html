Sesión 2: Conversión de Tipos, Comentarios y Estructuras de Control

1. Conversión de Tipos de Datos

La conversión de tipos (casting) permite cambiar un tipo de dato a otro.

1.1 Conversión a Entero (int)

Ejemplo 1:
numero_texto = "123"
numero_entero = int(numero_texto)
decimal = 45.8
entero_desde_decimal = int(decimal)
print(numero_entero) # 123
print(entero_desde_decimal) # 45
Ejemplo 2:
edad_str = "25"
edad_int = int(edad_str)
precio_float = 99.99
precio_int = int(precio_float)
print("Edad:", edad_int, type(edad_int))
Ejemplo 3:
verdadero = True
falso = False
bool_a_int_v = int(verdadero) # 1
bool_a_int_f = int(falso) # 0
print(bool_a_int_v, bool_a_int_f)

1.2 Conversión a Cadena (str)

Ejemplo 1:
edad = 25
edad_texto = str(edad)
precio = 99.99
precio_texto = str(precio)
print("Tengo " + edad_texto + " años")
Ejemplo 2:
activo = True
numero = 42
activo_str = str(activo)
numero_str = str(numero)
print("Estado: " + activo_str)
Ejemplo 3:
temperatura = 23.5
nivel = 8
temp_str = str(temperatura)
nivel_str = str(nivel)
mensaje = "Temperatura: " + temp_str + "°C, Nivel: " + nivel_str

1.3 Conversión a Decimal (float)

Ejemplo 1:
numero = 42
texto_decimal = "3.14"
numero_float = float(numero)
pi = float(texto_decimal)
print(numero_float) # 42.0
print(pi) # 3.14
Ejemplo 2:
edad = 25
precio_str = "199.99"
edad_float = float(edad)
precio_float = float(precio_str)
print(f"Edad: {edad_float}, Precio: {precio_float}")
Ejemplo 3:
verdadero = True
falso = False
bool_a_float_v = float(verdadero) # 1.0
bool_a_float_f = float(falso) # 0.0
print(bool_a_float_v, bool_a_float_f)

2. Comentarios en Python

Los comentarios son texto que Python ignora al ejecutar el código. Se usan para explicar el código.

2.1 Comentarios de una línea (#)

Ejemplo 1:
# Este es un comentario de una línea
nombre = "Juan" # Comentario al final de la línea
# La siguiente línea calcula la edad
edad = 2024 - 1995
Ejemplo 2:
# Configuración inicial del programa
precio_base = 100.0 # Precio sin descuentos
descuento = 0.15 # 15% de descuento
# precio_final = precio_base * (1 - descuento)
Ejemplo 3:
# Variables del juego
vidas_jugador = 3 # Número de vidas inicial
puntos = 0 # Puntos acumulados
# nivel_actual = 1 # Esta línea está comentada

2.2 Comentarios de múltiples líneas (""" """)

Ejemplo 1:
"""
Este es un comentario
de múltiples líneas
que explica el propósito del programa
"""
print("Hola mundo")
Ejemplo 2:
"""
Programa de cálculo de notas
Autor: Estudiante
Fecha: 2024
"""
nota1 = 8.5
nota2 = 9.0
promedio = (nota1 + nota2) / 2
Ejemplo 3:
'''
También se pueden usar comillas simples
para comentarios de múltiples líneas
'''
usuario = "admin"
contraseña = "123456"

3. Operadores

3.1 Operadores Aritméticos

Ejemplo 1:
a = 10
b = 3
suma = a + b # 13
resta = a - b # 7
multiplicacion = a * b # 30
division = a / b # 3.333...
print(f"Suma: {suma}, Resta: {resta}")
Ejemplo 2:
x = 15
y = 4
division_entera = x // y # 3 (división sin decimales)
modulo = x % y # 3 (resto de la división)
potencia = x ** y # 50625 (15 elevado a la 4)
print(f"División entera: {division_entera}, Módulo: {modulo}")
Ejemplo 3:
precio = 100
descuento = 0.20
impuesto = 0.21
precio_descuento = precio * (1 - descuento)
precio_final = precio_descuento * (1 + impuesto)
print(f"Precio final: ${precio_final}")

3.2 Operadores de Comparación

Ejemplo 1:
edad = 18
es_mayor = edad >= 18 # True
es_menor = edad < 18 # False
es_igual = edad == 18 # True
es_diferente = edad != 20 # True
print(f"¿Es mayor de edad? {es_mayor}")
Ejemplo 2:
nota1 = 8.5
nota2 = 7.0
nota1_mayor = nota1 > nota2 # True
notas_iguales = nota1 == nota2 # False
nota1_menor = nota1 <= nota2 # False
print(f"Nota1 es mayor: {nota1_mayor}")
Ejemplo 3:
precio_producto = 50
presupuesto = 75
puede_comprar = precio_producto <= presupuesto # True
es_caro = precio_producto > 100 # False
precio_exacto = precio_producto == 50 # True
print(f"¿Puede comprar? {puede_comprar}")

3.3 Operadores Lógicos

Ejemplo 1:
edad = 20
tiene_licencia = True
puede_conducir = edad >= 18 and tiene_licencia # True
es_joven = edad < 30 or edad > 65 # True
no_puede_votar = not (edad >= 18) # False
print(f"¿Puede conducir? {puede_conducir}")
Ejemplo 2:
llueve = False
hace_frio = True
mal_tiempo = llueve or hace_frio # True
buen_tiempo = not llueve and not hace_frio # False
clima_mixto = llueve and hace_frio # False
print(f"¿Mal tiempo? {mal_tiempo}")
Ejemplo 3:
es_estudiante = True
es_trabajador = False
tiene_descuento = es_estudiante or es_trabajador # True
trabaja_y_estudia = es_estudiante and es_trabajador # False
no_trabaja = not es_trabajador # True
print(f"¿Tiene descuento? {tiene_descuento}")

4. Estructuras Condicionales if...else

4.1 Condicional Simple (if)

Ejemplo 1:
edad = 20
if edad >= 18:
    print("Eres mayor de edad")
    print("Puedes votar")
Ejemplo 2:
temperatura = 30
if temperatura > 25:
    print("Hace calor")
    print("Usa ropa ligera")
Ejemplo 3:
nota = 8.5
if nota >= 7.0:
    print("¡Felicidades!")
    print("Has aprobado la materia")

4.2 Condicional if...else

Ejemplo 1:
edad = 16
if edad >= 18:
    print("Puedes votar")
else:
    print("Aún no puedes votar")
    print(f"Te faltan {18 - edad} años")
Ejemplo 2:
numero = 7
if numero % 2 == 0:
    print(f"{numero} es par")
else:
    print(f"{numero} es impar")
Ejemplo 3:
precio = 100
dinero = 80
if dinero >= precio:
    print("Puedes comprar el producto")
else:
    print("No tienes suficiente dinero")
    print(f"Te faltan ${precio - dinero}")

4.3 Condicional if...elif...else

Ejemplo 1:
nota = 8.5
if nota >= 9.0:
    print("Excelente")
elif nota >= 7.0:
    print("Bueno")
elif nota >= 5.0:
    print("Aprobado")
else:
    print("Reprobado")
Ejemplo 2:
edad = 25
if edad < 13:
    print("Eres un niño")
elif edad < 20:
    print("Eres un adolescente")
elif edad < 60:
    print("Eres un adulto")
else:
    print("Eres un adulto mayor")
Ejemplo 3:
temperatura = 22
if temperatura < 0:
    print("Está congelando")
elif temperatura < 15:
    print("Hace frío")
elif temperatura < 25:
    print("Temperatura agradable")
elif temperatura < 35:
    print("Hace calor")
else:
    print("Hace mucho calor")
