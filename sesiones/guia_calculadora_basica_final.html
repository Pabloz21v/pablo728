¿Qué debo saber para hacer una calculadora básica?

  1. Entrada y Salida Básicas:
    • Saber cómo recibir entrada del usuario usando input().
    • Saber cómo mostrar resultados usando print().
  2. Variables y Tipos de Datos:
    • Conocer los tipos de datos básicos como int (enteros) y float (números decimales).
    • Saber cómo convertir entre tipos de datos usando int() y float().
  3. Operadores Aritméticos:
    • Conocer los operadores básicos: + (suma), - (resta), * (multiplicación), / (división).
  4. Condicionales:
    • Saber cómo usar la declaración if para tomar decisiones en el código.

Ejemplo 1: Suma de dos números

# Pedir al usuario que ingrese dos números
num1 = float(input("Ingresa el primer número: "))
num2 = float(input("Ingresa el segundo número: "))

# Sumar los dos números
suma = num1 + num2

# Mostrar el resultado
print("La suma es:", suma)

Ejemplo 2: Uso del condicional if para decidir una operación

# Pedir al usuario que ingrese dos números
num1 = float(input("Ingresa el primer número: "))
num2 = float(input("Ingresa el segundo número: "))

# Pedir al usuario que elija una operación
operacion = input("¿Qué operación deseas realizar? (A si es una suma, B si es una resta): ")

# Realizar la operación basada en la entrada del usuario
if operacion == "A":
    resultado = num1 + num2
elif operacion == "B":
    resultado = num1 - num2
else:
    resultado = "Operación no válida"

# Mostrar el resultado
print("El resultado es:", resultado)

Ejemplo 3: Calculadora con múltiples operaciones

# Pedir al usuario que ingrese dos números
num1 = float(input("Ingresa el primer número: "))
num2 = float(input("Ingresa el segundo número: "))

# Pedir al usuario que elija una operación
print("MENÚ DE OPERACIONES:")
print("A. Suma (+)")
print("B. Resta (-)")
print("C. Multiplicación (*)")
print("D. División (/)")
operacion = input("Elija una operación (A, B, C, D): ")

# Realizar la operación basada en la entrada del usuario
if operacion == "A":
    resultado = num1 + num2
elif operacion == "B":
    resultado = num1 - num2
elif operacion == "C":
    resultado = num1 * num2
elif operacion == "D":
    if num2 != 0:
        resultado = num1 / num2
    else:
        resultado = "Error: División por cero"
else:
    resultado = "Operación no válida"

# Mostrar el resultado
print("El resultado es:", resultado)

Para realizar cálculos más complejos como el uso de Pi (3.14), necesitaremos escribir en la parte superior del fichero lo siguiente:

import math

¿Cuándo SÍ necesitas math?

Operación Función en math
Logaritmos log(), log10()
Trigonometría sin(), cos()
Constantes (π, e) math.pi, math.e
Raíz cuadrada exacta math.sqrt()
Factoriales math.factorial() 
import math # Importar módulo para operaciones avanzadas

print("Calculadora Científica")
opcion = input("Ingrese “A” para calcular una raiz: ")

# Raíz cuadrada
if opcion == "A":
    num = float(input("Número para raíz cuadrada: "))
    if num < 0:
        resultado = "¡Error! Número negativo."
    else:
        resultado = math.sqrt(num)
else:
    resultado = "Operación no soportada."

print("Resultado:", resultado)

Nota a observar, siempre colocar la variable que se quiere calcular dentro de los paréntesis.

Ej: resultado = math.sqrt(num)

Dentro de la variable “num” se encuentra el valor que seleccionaron para su cálculo.

Para el caso de Pi

import math

radio = float(input("Ingrese el radio de la esfera: "))
volumen = (4/3) * math.pi * (radio ** 3)
print(f"Volumen: {volumen:.2f} de un radio de {radio}")

f permite agregar una variable dentro de las comillas de formas más fácil.

.2f permite redondear el número con 2 números decimales.

Ej: 2599.85
