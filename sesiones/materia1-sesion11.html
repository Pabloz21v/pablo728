Sesión 11: LEY 24.766 (PROPIEDAD INTELECTUAL) Y LEY 26.388 (DELITOS INFORMÁTICOS)

1. LEY 24.766 - CONFIDENCIALIDAD SOBRE INFORMACIÓN Y PRODUCTOS

1.1 Marco General

La Ley 24.766, sancionada en 1996, establece el régimen legal aplicable a la protección de la información confidencial no divulgada y los datos de prueba u otros datos cuya elaboración suponga un esfuerzo considerable.

1.2 Información Confidencial

Se considera información confidencial aquella que:

  • Sea secreta (no conocida o fácilmente accesible)
  • Tenga valor comercial por ser secreta
  • Haya sido objeto de medidas razonables para mantenerla secreta

1.3 Protección del Software

El software se protege como obra intelectual bajo la Ley 11.723, pero la Ley 24.766 complementa esta protección para:

  • Código fuente no divulgado
  • Algoritmos propietarios
  • Bases de datos con valor comercial
  • Documentación técnica confidencial

2. DERECHOS DE AUTOR SOBRE BASES DE DATOS

2.1 Bases de Datos como Obra Intelectual

Las bases de datos están protegidas cuando representen una creación intelectual por la selección o disposición de su contenido. La protección abarca:

  • La estructura y organización
  • Los criterios de selección
  • La forma de presentación
  • No necesariamente los datos individuales

2.2 Derechos del Creador

El titular de una base de datos tiene derecho a:

  • Autorizar o prohibir la reproducción total o parcial
  • Controlar la distribución y comunicación pública
  • Autorizar transformaciones o adaptaciones
  • Recibir remuneración por el uso comercial

3. LEY 26.388 - DELITOS INFORMÁTICOS

3.1 Modificaciones al Código Penal

La Ley 26.388 de 2008 modificó el Código Penal argentino para incorporar los delitos informáticos, actualizando tipos penales tradicionales para el entorno digital.

3.2 Acceso Ilícito a Sistemas Informáticos

Artículo 153 bis: "Será reprimido con prisión de quince días a seis meses, si no resultare un delito más severamente penado, el que a sabiendas accediere por cualquier medio, sin la debida autorización o excediendo la que posea, a un sistema o dato informático de acceso restringido."

Elementos del delito:

  • Acceso a sistemas informáticos
  • Sin autorización o excediendo la autorización
  • Sistema de acceso restringido
  • Conocimiento de la restricción

3.3 Daño Informático

Artículo 183: Se incluye como daño la alteración, destrucción o inutilización de datos, documentos, programas o sistemas informáticos.

Modalidades agravadas:

  • Daño a sistemas informáticos de servicios públicos
  • Interrupción o entorpecimiento de comunicaciones
  • Daño que afecte el funcionamiento de un sistema informático

3.4 Fraude Informático

Artículo 173 inc. 16: "El que defraudare mediante el uso de una tarjeta de compra, crédito o débito, cuando la misma hubiere sido falsificada, clonada, o cuando se utilizaren los datos de una tarjeta legítima."

4. VIOLACIÓN DE SECRETOS Y PRIVACIDAD

4.1 Interceptación de Comunicaciones

Artículo 153: "Será reprimido con prisión de quince días a seis meses el que abriere o accediere indebidamente a una comunicación electrónica, una carta, un pliego cerrado, un despacho telegráfico, telefónico o de otra naturaleza, que no le esté dirigido."

4.2 Captación de Comunicaciones

Se penaliza la captación dolosa e ilegítima de comunicaciones entre terceros, incluyendo:

  • Interceptación de emails
  • Captura de comunicaciones en redes
  • Acceso a mensajería instantánea
  • Interceptación de datos en tránsito

5. CASOS PRÁCTICOS Y PRECEDENTES

Caso 1: Empleado Desleal

Un programador que trabaja en una empresa de software copia el código fuente de un sistema propietario antes de renunciar. Posteriormente, utiliza este código para desarrollar un producto competidor en su nueva empresa.

Análisis legal:

  • Violación de confidencialidad (Ley 24.766)
  • Posible daño informático
  • Competencia desleal
  • Responsabilidad civil y penal

Caso 2: Hackeo de Base de Datos

Un grupo de hackers accede ilegalmente a la base de datos de una empresa de tarjetas de crédito, extrayendo información de 50.000 clientes. Posteriormente, venden esta información en mercados ilegales de internet.

Delitos aplicables:

  • Acceso ilícito a sistema informático (Art. 153 bis)
  • Violación de secretos (Art. 153)
  • Fraude informático (Art. 173 inc. 16)
  • Asociación ilícita

Caso 3: Piratería de Software

Una empresa utiliza software sin licencia en todas sus computadoras. Además, modifica el código para eliminar las restricciones de uso y distribuye copias modificadas a empresas subsidiarias.

Infracciones identificadas:

  • Violación de derechos de autor
  • Modificación no autorizada
  • Distribución ilegal
  • Daño patrimonial al titular
