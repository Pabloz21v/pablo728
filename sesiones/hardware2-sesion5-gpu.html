Conceptos Clave

1. Arquitecturas Modernas

NVIDIA Blackwell (RTX 50XX Series):

  • Proceso: TSMC 3nm (N3).
  • Núcleos RT 4ta gen: +3x eficiencia ray tracing vs Ada Lovelace.
  • Núcleos Tensor 5ta gen: +6x rendimiento AI vs Ada, soporte para DLSS 4.0.
  • DLSS 4.0: Frame Generation mejorada + Super Resolution con IA de transformación.
  • AV1 encoding: Hardware dedicado para streaming 8K.
  • Modelos: RTX 5060, 5070, 5080, 5090.

AMD RDNA 4 (RX 9000 Series):

  • Proceso: TSMC 4nm + 6nm chiplet.
  • Ray Accelerators 3ra gen: +2x eficiencia ray tracing vs RDNA 3.
  • AI Accelerators: Mejorados para FSR 4.0 (IA acelerada por hardware).
  • FSR 4.0: Frame Generation con calidad indistinguible de DLSS 4.0, compatible con cualquier GPU.
  • AV1/H.264 dual encoders: Streaming y grabación mejorados.
  • Modelos: RX 9600, 9700 XT, 9800 XT, 9900 XTX.

Intel Arc (Battlemage):

  • Proceso: TSMC 6nm.
  • XeSS 1.3: Upscaling con IA, compatible con GPUs Intel/AMD/NVIDIA.
  • Modelos: Arc B580, B780 (gama media-baja).

3. Ray Tracing y Tecnologías de Aceleración

  • Ray Tracing: Simulación realista de luz mediante trazado de rayos.
  • RT Cores (NVIDIA 4ta gen): Hardware especializado para intersección rayo-triángulo.
  • Ray Accelerators (AMD 3ra gen): Unidades similares en RDNA 4.
  • Performance Cost: 20-30% pérdida FPS sin upscaling (mejorado vs 2024).
  • Efectos: Reflecciones, sombras, iluminación global, oclusión ambiental.

4. DLSS 4.0 vs FSR 4.0 - Upscaling Technologies

Característica DLSS 4.0 (NVIDIA) FSR 4.0 (AMD)
Método IA/ML (Tensor Cores 5ta gen) Temporal + IA (AI Accelerators)
Frame Generation Sí (hasta 3x frames generados) Sí (calidad similar a DLSS 4.0)
Compatibilidad Solo RTX 20XX+ Cualquier GPU (AMD/NVIDIA/Intel)
Calidad Superior (especialmente 4K) Paridad con DLSS 4.0
Latencia NVIDIA Reflex integrado AMD Anti-Lag+

5. Núcleos de Procesamiento

NVIDIA:

  • CUDA Cores: Procesadores de shaders estándar.
  • RT Cores 4ta gen: Ray tracing (Blackwell).
  • Tensor Cores 5ta gen: AI/ML y DLSS 4.0.

AMD:

  • Stream Processors: Equivalente a CUDA Cores.
  • Ray Accelerators 3ra gen: Ray tracing hardware.
  • AI Accelerators: Mejorados para FSR 4.0 y cómputo paralelo.

Tipos de Uso y Características Esenciales

Ofimática/Navegación (iGPU suficiente)

Objetivo:

Display output básico, video playback, trabajo de oficina.

Características Críticas (🟢):

  • GPU integrada suficiente (Intel UHD, AMD Vega, Apple M-series).
  • Soporte múltiples monitores si es necesario.
  • Decodificación H.264/H.265 hardware.
  • Bajo consumo energético.

Irrelevantes (🔴):

  • Ray tracing, DLSS/FSR.
  • Gaming performance.
  • VRAM alta capacidad.
  • Overclocking capabilities.

Gaming 1080p (Entry-Mainstream)

Objetivo:

60+ FPS en títulos AAA, 120+ FPS en competitivos.

Características Críticas (🟢):

  • GPU: GTX 1660 Super+ / RX 6500 XT+ mínimo.
  • VRAM: 6 (GB+ para títulos modernos.
  • Soporte DirectX 12 / Vulkan.
  • TGP: 100-180W (PSU 550W+).

Características Útiles (🟡):

  • DLSS/FSR para extended lifespan.
  • Ray tracing básico si soportado.
  • 1-2 generaciones atrás para mejor valor.

Ejemplo Gaming 1080p:

  • GPU: RTX 5060 / RX 9600 (~$300 USD / $360,000 ARS).
  • VRAM: 8GB GDDR6.
  • Performance: 80+ FPS títulos AAA con high settings.
  • Ray tracing: Usable con DLSS/FSR.

Gaming 1440p (Mainstream-High End)

Objetivo:

60+ FPS máximos settings, ray tracing selectivo.

Características Críticas (🟢):

  • GPU: RTX 5070 + / RX 9700 XT+ recomendado.
  • VRAM: 12GB+ para futuro-proofing.
  • DLSS/FSR soporte para ray tracing.
  • TGP: 200-300W (PSU 650-750W).

Ejemplo Gaming 1440p:

  • GPU: RTX 5070 Super / RX 9700 XT.
  • VRAM: 12-16GB.
  • Performance: 70+ FPS máximos settings, 90+ FPS con DLSS Quality.
  • Ray tracing: Buena experience con upscaling.

Gaming 4K (High End-Enthusiast)

Objetivo:

60+ FPS máximos settings con ray tracing.

Características Críticas (🟢):

  • GPU: RTX 5080 + / RX 9800 XT+ mínimo.
  • VRAM: 16GB+ obligatorio.
  • DLSS/FSR crítico para performance.
  • TGP: 350W+ (PSU 850W+ recomendado).

Ejemplo Gaming 4K:

  • GPU: RTX 4090 (~$1600 USD / $1,920,000 ARS).
  • VRAM: 24GB GDDR6X.
  • Performance: 60+ FPS nativos, 100+ FPS con DLSS Performance.
  • Ray tracing: Máximos settings usable.

Edición de Video (4K/8K)

Objetivo:

Encoding acelerado, timeline scrubbing fluido, effects rendering.

Características Críticas (🟢):

  • VRAM: 8GB+ para 4K, 16GB+ para 8K.
  • Hardware encoders: NVENC (NVIDIA) o AMF (AMD).
  • CUDA Cores/Stream Processors para effects.
  • Soporte AV1 encoding para streaming moderno.

Características Útiles (🟡):

  • Multiple encoders simultáneos
  • 10-bit pipeline support
  • OpenCL/CUDA acceleration

Ejemplo Edición 4K:

  • GPU: RTX 4070 / RX 7700 XT (~$500 USD / $600,000 ARS).
  • Software: Premiere Pro, DaVinci Resolve.
  • Beneficio: 3-5x faster export vs CPU-only.

Workstation IA/ML

Objetivo:

Training de modelos, inference acelerado, compute workloads.

Características Críticas (🟢):

  • VRAM: 16GB+ mínimo, 24GB+ recomendado.
  • Tensor Cores/AI Accelerators para ML frameworks.
  • CUDA/ROCm software stack maduro.
  • ECC memory en workstation cards.
  • Multi-GPU scaling capabilities.

Ejemplo Workstation IA:

  • GPU: RTX 4090 / RTX A6000 (~$1600-4000 USD).
  • Frameworks: PyTorch, TensorFlow, CUDA.
  • Models: Stable Diffusion, LLaMA, custom training.

Mining/Compute

Objetivo: Hash rate, eficiencia energética, ROI calculation.

Factores Clave:

  • Hash rate por watt (efficiency)
  • Memoria bandwidth para memory-hard algorithms
  • Availability vs gaming market impact
  • LHR (Lite Hash Rate) limitations en gaming cards

Requerimientos de PSU

Cálculo recomendado:

TGP GPU + TDP CPU + 150W overhead del sistema.

  • Conectores: Verificar 8-pin PCIe vs nuevo 12VHPWR high-end (16-pin).
  • Eficiencia: 80+ Gold mínimo para GPUs high-end.
  • Calidad: Tier A/B en PSU tier list para componentes premium.

PCIe Compatibilidad

  • PCIe 5.0 x16: Introduciendoce, backward compatible.
  • PCIe 4.0 x16: Estándar actual, backward compatible.
  • PCIe 3.0 x16: Compatible, pérdida mínima performance (<5%).
  • PCIe 4.0 x8: Posible bottleneck en GPUs high-end.
  • Resizable BAR/Smart Access Memory: 2-8% ganancia en títulos compatibles.
