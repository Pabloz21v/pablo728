Clase 14: Condicional Múltiple (elif) y Condiciones Compuestas

Marco Teórico

Cuando existen más de dos alternativas posibles, usar solo if / else puede volverse insuficiente o poco claro. Python ofrece elif (abreviatura de "else if") para encadenar múltiples condiciones. Solo se ejecuta el bloque de la primera condición que resulte verdadera; el resto se saltea.

Además, podemos construir condiciones compuestas combinando comparaciones con los operadores lógicos and, or y not, cerrando el círculo entre lógica proposicional y programación.

Sintaxis

if condicion_1:
    # bloque 1

elif condicion_2:
    # bloque 2

elif condicion_3:
    # bloque 3

else:
    # bloque por defecto (ninguna condición se cumplió)

Ejemplos

Ejemplo 1 — Calificaciones por letra:

nota = float(input("Ingresá tu nota (0-10): "))

if nota >= 9:
    print("Sobresaliente 🏆")
elif nota >= 7:
    print("Muy bueno 👍")
elif nota >= 6:
    print("Aprobado ✓")
else:
    print("Desaprobado ✗")

Ejemplo 2 — Condición compuesta con and:

edad = int(input("Edad: "))
permiso = input("¿Tenés permiso de tus padres? (si/no): ")

if edad >= 16 and permiso == "si":
    print("Podés participar del viaje de egresados.")
else:
    print("No podés participar.")

Ejemplo 3 — Condición con or:

dia = input("¿Qué día es hoy? ").lower()

if dia == "sabado" or dia == "domingo":
    print("¡Es fin de semana! 🎉")
else:
    print("Es día de semana, a estudiar 📚")

Ejemplo 4 — Menú de opciones:

print("=== Calculadora ===")
print("1 - Sumar")
print("2 - Restar")
opcion = input("Elegí una opción: ")

if opcion == "1":
    a = float(input("Primer número: "))
    b = float(input("Segundo número: "))
    print("Resultado:", a + b)
elif opcion == "2":
    a = float(input("Primer número: "))
    b = float(input("Segundo número: "))
    print("Resultado:", a - b)
else:
    print("Opción no válida.")

💡 Tip para el docente:

Retomar las tablas de verdad de la Semana 1 y mostrar cómo and y or en Python funcionan exactamente igual que los conectores lógicos. Este puente entre teoría y práctica refuerza la comprensión de ambos bloques. Se puede proponer que conviertan una tabla de verdad en un if.

Actividad 14: Programas con elif y condiciones compuestas

  1. Pedir la nota del estudiante (0-10) y clasificarla: "Sobresaliente" (9-10), "Muy Bueno" (7-8), "Aprobado" (6), "Desaprobado" (0-5).
  2. Clasificar la temperatura: si es menor a 0°C → "Bajo cero 🥶", entre 0 y 15 → "Frío 🧥", entre 16 y 25 → "Agradable ☀️", mayor a 25 → "Calor 🌞".
  3. Pedir el número de un mes (1-12) y mostrar a qué trimestre del año pertenece (1-3: Primer trimestre, etc.).
  4. Pedir edad y mostrar la categoría deportiva: "Infantil" (8-11), "Cadete" (12-14), "Junior" (15-17), "Senior" (18 o más). Si es menor de 8: "Sin categoría aún".
  5. Sistema de login: pedir usuario y contraseña. Solo si usuario es "admin" y contraseña es "1234" mostrar "Sesión iniciada". En cualquier otro caso, "Credenciales incorrectas".
  6. Pedir un número entre 1 y 7 y mostrar el día de la semana correspondiente (1→Lunes, ..., 7→Domingo). Si el número no está en ese rango, mostrar "Número inválido".
  7. Calculadora básica con menú: pedir dos números y una operación (+, -, *, /). Usar elif para cada operación. Controlar la división por cero.
  8. Pedir el sueldo de una persona. Si gana menos de $50.000, no paga impuesto. Si gana entre $50.000 y $100.000, paga 9%. Si gana más de $100.000, paga 17%. Calcular e imprimir el impuesto.
  9. Pedir si hace frío (si/no) y si tiene campera (si/no). Usando and / or, indicar: si hace frío y no tiene campera → "¡Abrigate!"; si hace frío y tiene campera → "Estás bien"; si no hace frío → "No necesitás campera".
  10. Desafío: Pedir tres números distintos e indicar cuál es el mayor, cuál el del medio y cuál el menor, sin usar funciones de Python (solo if / elif / else).
