Sesión 5: Virus informáticos y malware

1. Conceptos Clave: Virus y Malware

En el lenguaje cotidiano solemos llamar "virus" a cualquier problema de seguridad en una computadora, pero en realidad no todo programa malicioso es un virus. Es fundamental establecer una distinción técnica precisa:

  • Virus informático: Es un tipo específico de software malicioso que necesita "infectar" otros archivos o programas para ejecutarse y propagarse. Su nombre proviene de la biología: al igual que un virus biológico necesita una célula para multiplicarse, un virus informático necesita un archivo huésped o un sistema donde alojarse.
  • Malware (Malicious Software): Es el término general y técnico que engloba a cualquier software creado con intención dañina o invasiva.
🧠 Idea clave No todo malware es un virus, pero todo virus sí forma parte del conjunto de malware. Dentro de esta familia encontramos: virus, gusanos, troyanos, spyware, ransomware, keyloggers, rootkits, adware, entre otros. Aprender a diferenciarlos permite diagnosticar mejor y evitar soluciones improvisadas.

2. Tipos Principales de Malware

Cada tipo de amenaza entra de forma distinta, deja huellas diferentes y exige respuestas técnicas específicas.

Tipo Cómo actúa Ejemplo cotidiano Riesgo o Daño principal
Virus Se adjunta a archivos o programas y se activa al ejecutarlos. Un archivo infectado compartido por pendrive. Corrupción de documentos o borrado de archivos.
Gusano (Worm) Se propaga y replica automáticamente por redes sin intervención del usuario. Una amenaza que salta de una PC del laboratorio a otra. Saturación de redes escolares o empresariales (propagación masiva).
Troyano Se disfraza de programa útil o legítimo para engañar al usuario. Un "crack" para activar un juego o programa. Robo de datos, contraseñas bancarias o apertura de "puertas traseras".
Spyware Espía la actividad del usuario sin su consentimiento. Software oculto que registra los sitios web visitados. Pérdida de privacidad y captura de historial.
Keylogger Registra todas las teclas presionadas por el usuario. Programa invisible que captura lo escrito en el teclado. Robo de usuarios y contraseñas.
Ransomware Cifra los archivos del usuario y exige un pago para recuperarlos. Documentos escolares o fotos que ya no abren y piden rescate. Secuestro de la información y bloqueo de datos.
Adware Muestra publicidad invasiva o redirecciona el navegador. Ventanas emergentes que aparecen solas. Molestia constante, rastreo y posible descarga de otras amenazas.

3. ¿Cómo se propagan las amenazas?

Una idea equivocada es pensar que las amenazas "entran solas". Casi siempre existe una combinación de tres factores: vulnerabilidad técnica, descuido humano y falta de prevención. Las vías de propagación más comunes incluyen:

  1. Archivos adjuntos en correos falsos (Phishing): Correos que simulan provenir de una entidad real para engañar al usuario.
  2. Descargas desde sitios no confiables: Incluye software pirata, instaladores dudosos y webs de dudosa reputación.
  3. Pendrives y discos externos infectados: Especialmente peligrosos si se conectan en múltiples equipos (hogar, escuela, cibercafé).
  4. Macros maliciosas: Scripts ocultos dentro de documentos ofimáticos (Word, Excel).
  5. Explotación de fallos del sistema: Ocurre cuando el sistema operativo o el software no están actualizados.
  6. Redes compartidas mal configuradas: Permiten el paso de gusanos entre equipos conectados.
  7. Enlaces falsos por mensajería: Sorteos, premios o urgencias inventadas que llegan por chat.
  8. Uso de "Cracks" o activadores: Programas para usar software de pago gratis que frecuentemente incluyen troyanos.

4. Síntomas de una Infección

Aunque algunas amenazas están diseñadas para ser invisibles, existen señales de alerta que un técnico debe reconocer:

  • Aparición de ventanas emergentes o publicidades extrañas.
  • Creación de accesos directos desconocidos.
  • El navegador cambia su página de inicio sin permiso.
  • Archivos que desaparecen, cambian de nombre o quedan inaccesibles.
  • Uso excesivo de disco, red o procesador sin explicación aparente.
  • El antivirus se desactiva solo o no permite abrirse.
⚠️ Muy importante: No toda lentitud implica virus. Una PC puede ir lenta por falta de memoria, disco lleno o demasiados programas abiertos. El técnico no debe diagnosticar "a ojo", sino observar, comparar síntomas y usar herramientas.

5. Impacto sobre los Datos

El malware afecta directamente a los pilares de la seguridad informática: integridad, disponibilidad y confidencialidad.

Dimensión afectada ¿Qué significa? Ejemplo de daño
Integridad Los datos fueron modificados, alterados o dañados sin autorización. Un archivo cambia su contenido o una base de datos es corrompida.
Disponibilidad Los datos existen, pero no pueden usarse cuando se necesitan. Ransomware que cifra documentos impidiendo su apertura.
Confidencialidad La información llega a personas no autorizadas. Robo de contraseñas, fotos privadas o listados de clientes.

6. Ejemplos y Casos Cotidianos

Para contextualizar el aprendizaje, analicemos situaciones comunes:

El "juego gratis" (Troyano): Si descargas un juego desde una web no oficial y, al instalarlo, se introduce en segundo plano un programa que roba contraseñas, probablemente no sea un virus clásico, sino un troyano. El usuario creyó instalar algo útil, pero abrió la puerta a una amenaza.

El editor de vídeo "Premium" (Cadena de errores): Un estudiante descarga una versión pirateada de un editor de vídeo para un trabajo práctico. El instalador funciona, pero instala un troyano silencioso. Días después, su correo envía spam y pierde el acceso a sus redes sociales. La infección no fue "mala suerte", sino el resultado de decisiones inseguras.

La factura falsa (Phishing/Ransomware): Una alumna abre un archivo adjunto que decía ser una factura. A los pocos minutos, su fondo de pantalla cambia y sus documentos no abren (han sido cifrados). Esto es un ataque típico de ransomware iniciado por ingeniería social.
