Clase 11: Entrada y Salida (Input/Print)

Marco Teórico

Para que el programa sea interactivo, necesitamos pedir datos al usuario.

La función input() se utiliza para recibir datos del usuario. Siempre devuelve un texto (string), aunque el usuario escriba un número.

Ejemplo:

variable = input("Mensaje para el usuario: ")

Ejemplo

nombre = input("¿Cómo te llamas? ") print("Hola", nombre)

Actividad 11: Programas interactivos

Crea un archivo .py para cada uno de los siguientes ejercicios y ejecuta el código:

  1. Pedir al usuario su nombre y saludarlo.
  2. Pedir el nombre de su mascota e imprimirlo.
  3. Pedir el nombre y el apellido por separado, y mostrar "Bienvenido [Apellido], [Nombre]".
  4. Pedir la ciudad donde vive y mostrar "Vives en [ciudad]".
  5. Preguntar "¿Cuál es tu comida favorita?" y repetir la respuesta.
  6. Pedir tres palabras al usuario y mostrarlas en orden inverso (primero mostrar la 3ra, etc).
  7. Hacer un programa que simule un formulario: pedir Nombre, Edad, Mail y mostrar todo al final.
  8. Pedir al usuario que complete la frase "Mi deporte favorito es...".
  9. Pedir un color y mostrar "Tu color elegido es [color]".
  10. Usar input() sin guardar en variable (solo para pausar el programa "Presione Enter para salir").
