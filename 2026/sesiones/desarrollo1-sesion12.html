🧮 Clase 12: Tipos de datos y Operadores Aritméticos

Aprende a diferenciar textos de números y a realizar cálculos con Python

🔄 Repaso previo En la Clase 11 aprendimos a comunicarnos con el usuario a través de la función input(). Vimos que input() tiene una característica especial: siempre devuelve la información en formato de texto (string), incluso si el usuario teclea números (podemos decir que lo recibido desde consola lo guarda entre comillas "hola"). Hoy vamos a profundizar en qué significa esto de "formato de texto" frente a "formato numérico" y cómo podemos realizar operaciones matemáticas con ellos.

🎯 Objetivos de la clase

  • Comprender en detalle los tipos de datos: enteros (int), decimales (float) y cadenas de texto (str).
  • Utilizar la función type() para verificar el tipo de cualquier dato.
  • Distinguir la diferencia crítica entre un número real (ej. 25) y un texto que parece número (ej. "25").
  • Aprender a utilizar los operadores aritméticos básicos en Python.
  • Entender el orden de las operaciones matemáticas.

📦 Qué son los tipos de datos

Imagina que estás organizando una mudanza. Tienes cajas para libros, bolsas para ropa y fundas para objetos frágiles. Cada tipo de objeto requiere un contenedor diferente y se maneja de forma distinta. En programación, sucede lo mismo: el ordenador necesita saber exactamente qué tipo de información está guardando en una variable para saber qué operaciones puede hacer con ella. ¿Puede sumar estas dos cosas? ¿Puede ponerlas en mayúsculas? Depende de su tipo.

1. Enteros (int)

Son números sin parte decimal, tanto positivos como negativos, incluyendo el cero. Su nombre viene de integer. 

edad = 25
temperatura = -5
año = 2024

2. Decimales (float)

Son números que tienen una parte fraccionaria (número con coma). Su nombre viene de floating point (coma flotante).

⚠️ ¡Importante! El separador decimal

En español solemos usar la coma para los decimales (ej. 3,14), pero en Python y en la mayoría de lenguajes de programación, se debe utilizar el punto (.).

✅ Correcto: 3.14 usamos el punto
❌ Incorrecto: 3,14no usemos la coma

precio = 19.99
peso = 75.5
pi = 3.14159

3. Cadenas de texto (str)

Representan secuencias de caracteres: letras, símbolos, e incluso números si los tratamos como texto. Su nombre viene de string. Siempre deben ir entre comillas simples ('...') o dobles ("..."). 

nombre = "Laura"
mensaje = '¡Hola a todos!'
numero_falso = "42"  # ¡Ojo! Al tener comillas, esto es texto, no un número para operar.

🔍 La función type()

A veces, mirando el código, podemos tener dudas sobre qué tipo de dato hay en una variable. Python nos ofrece un "detector de mentiras" llamado type(). Esta función nos devuelve el tipo exacto del dato que le pasemos. 

numero = 10
print(type(numero))  # <class 'int'>
# observen como al guardar el dato en la variable SIN usar las comillas este SI lo interpreta como número.
# Y lo confirmamos ya que print(type(numero)) nos entrega 'int' ósea número entero.
# Muy importante para realizar cálculos matemáticos.

numero_dos = "10"
print(type(numero_dos))  # <class 'str'>
# observen como al guardar el dato en la variable CON usar las comillas este SI lo interpreta como texto.
# Y lo confirmamos ya que print(type(numero_dos)) nos entrega 'str' ósea una cadena de texto.

numero_tres = 10.0
print(type(numero_tres))  # <class 'float'>
# observen como al guardar el dato en la variable SIN usar las comillas este SI lo interpreta como número decimal.
# Y lo confirmamos ya que print(type(numero_tres)) nos entrega 'float' ósea número decimal.
# Muy importante para realizar cálculos matemáticos.

También puedemos usar type() directamente sobre un valor, sin necesidad de guardarlo antes en una variable. Esto resulta muy útil cuando estás practicando y quieres comprobar rápidamente cómo está interpretando Python lo que has escrito.

Expresión Qué ve Python Tipo devuelto por type()
type(8) Un número entero int
type(8.0) Un número decimal float
type("8") Un texto formado por un carácter str
type("Hola") Una cadena de texto str

🎭 Número vs. Texto:

Este es el punto donde ocurren el 90% de los errores al empezar a programar. Observa estos dos casos:

<># Caso A: Suma de enteros (int)
primer_numero = 10
segundo_numero = 5
print(primer_numero + segundo_numero)  # Resultado: 15

# Caso B: Suma de textos (str)
texto_uno= "10"
texto_dos = "5"
print(texto_uno + texto_dos)  # Resultado: 105

# Caso C: Suma de textos (str)
texto_tres = "tomá"
texto_cuatro = "té"
print(texto_tres + texto_cuatro)  # Resultado: tomáté

¿Qué ha pasado en el Caso B? Cuando usamos el operador + con números, Python los suma matemáticamente. Pero cuando usamos el operador + con textos (strings), Python los concatena, es decir, los pega uno detrás del otro como vagones de tren. ¡Por eso "10" + "5" resulta en "105"!
Caso C: si sumamos textos ingresando letras en vez de números se concatenaran los textos uniendolos dando por resultado tomáté.

🧮 Operadores Aritméticos

Ahora que ya sabemos distinguir los números verdaderos, veamos qué podemos hacer con ellos:

Operador Nombre Ejemplo de uso Resultado esperado
+ Suma 10 + 3 13
- Resta 10 - 3 7
* Multiplicación 10 * 3 30
/ División exacta 10 / 3 3.3333333333333335 (siempre devuelve float)
// División entera 10 // 3 3 (descarta los decimales)
% Módulo (resto) 10 % 3 1 (lo que sobra de dividir 10 entre 3)
** Potencia (exponente) 10 ** 3 1000 (10 al cubo: 10 * 10 * 10)

El Orden de las Operaciones

Python sigue las mismas reglas matemáticas que aprendiste en la escuela para resolver ecuaciones combinadas
Es importante saber este orden para lograr cálculos correctos.

  1. Paréntesis ()
  2. Exponentes **
  3. Multiplicación, División y Módulo * / // % (de izquierda a derecha)
  4. Suma y Resta + - (de izquierda a derecha)
resultado = 10 + 5 * 2
# Primero multiplica (5*2=10), luego suma (10+10=20)
print(resultado)  # 20

resultado2 = (10 + 5) * 2
# Primero el paréntesis (10+5=15), luego multiplica (15*2=30)
print(resultado2) # 30

💡 Ejemplos guiados paso a paso (ejecutarlos en el VSC)

📌 Ejemplo 1: Verificar tipos de datos

nombre = "Marta"
edad = 19
altura = 1.68

print(nombre)
print(type(nombre))
print(edad)
print(type(edad))
print(altura)
print(type(altura))

Interpretación línea por línea: primero guardamos un texto en nombre, luego un entero en edad y después un decimal en altura. Cada vez que usamos type(variable), Python nos muestra cómo está clasificando internamente ese dato.

📌 Ejemplo 2: La diferencia entre sumar y concatenar

numero1 = 10
numero2 = 5
texto1 = "10"
texto2 = "5"

print(numero1 + numero2)  # 15
print(texto1 + texto2)    # 105

Interpretación: en las dos primeras variables hay números reales, así que el operador + realiza una suma matemática. En las dos últimas variables hay textos entre comillas, por eso el mismo operador + no suma, sino que pega un texto detrás del otro.

📌 Ejemplo 3: Mini cuenta con varios operadores

precio = 12.5
cantidad = 4
subtotal = precio * cantidad
promedio = subtotal / 2
resto = 13 % 5
potencia = 2 ** 3

print("Subtotal:", subtotal)
print("Promedio:", promedio)
print("Resto:", resto)
print("Potencia:", potencia)

Interpretación línea por línea: multiplicamos precio por cantidad para calcular un subtotal. Después dividimos ese subtotal entre 2. Luego calculamos el resto de dividir 13 entre 5 con %. Por último, usamos ** para calcular 2 elevado a 3. Este ejemplo te muestra que los operadores aritméticos pueden combinarse para resolver situaciones reales.

✨ Buenas prácticas

Aunque conozcan el orden de operaciones, es muy recomendable usar paréntesis para que nuestro código sea más fácil de leer para otros humanos. Es mejor escribir (10 * 5) + 3 que 10 * 5 + 3. Además, usar nombres de variables claros como precio_total o cantidad_productos para entender los cálculos con facilidad.

❌ Errores Frecuentes

  • Usar coma decimal en lugar de punto: pi = 3,14 creará un tipo de dato diferente (una tupla, lo verán mas adelante) en lugar de un float. Recuerda usar 3.14.
  • Poner comillas a números sin querer: Escribir edad = "25" en lugar de edad = 25. Luego, al querer sumar años, te dará error de tipos.
  • Intentar sumar texto con número: "Tengo " + 25 + " años". Python no sabe si quieres sumar o concatenar, y lanzará un TypeError. (Se soluciona con f-strings como vimos en clases pasadas).
  • Confundir / con //: 10 / 2 da 5.0 (float), mientras que 10 // 2 da 5 (int).

📌 Resumen de la clase

  • Los tipos básicos que manejamos son: enteros (int), decimales (float) y texto (str).
  • La función type(variable) nos ayuda a descubrir cómo Python interpreta un dato concreto.
  • ¡Cuidado con las comillas! 15 es un número con el que podemos operar, "15" es puro texto.
  • El operador + suma números, pero concatena (pega) textos.
  • Tenemos operadores para suma (+), resta (-), multiplicación (*), división exacta (/), división entera (//), módulo o resto (%) y potencia (**).
  • Las operaciones siguen el orden matemático tradicional, priorizando siempre los paréntesis.
