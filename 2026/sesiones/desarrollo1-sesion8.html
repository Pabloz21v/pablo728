Clase 8: Pseudocódigo y Diagramación

Marco Teórico

El Pseudocódigo es una forma de escribir algoritmos utilizando un lenguaje mezcla entre el lenguaje humano y el de programación. No tiene reglas de sintaxis estrictas, pero debe ser claro. Se utilizan palabras clave como: LEER, IMPRIMIR, CALCULAR, SI/ENTONCES.

Ejemplo 1: Área de un triángulo


    INICIO

        LEER base 

        LEER altura 

        area = (base * altura) / 2 

        IMPRIMIR area 

    FIN
INICIO: comienza el algoritmo.
Las líneas entre medio son las instrucciones que se ejecutan en orden.
base: la base del triangulo.
altura: altura del triangulo.
area: resultado de calcular base por altura dividido dos.
IMPRIMIR: muestra el resultado del calculo
FIN: termina el algoritmo.

Ejemplo 2: Calcular el promedio de tres notas

Problema: Solicitar tres notas al usuario y mostrar el promedio. 


    INICIO 
        LEER nota1 
        LEER nota2 
        LEER nota3 
        suma = nota1 + nota2 + nota3 
        promedio = suma / 3 
        IMPRIMIR promedio 
    FIN

Explicación: Se leen las tres notas, se suman, se dividen entre 3 para obtener el promedio y finalmente se muestra el resultado.

Ejemplo 3: Determinar si una persona es mayor de edad

Problema: Leer la edad de una persona y mostrar si es mayor de edad (18 años o más). 


    INICIO 
        LEER edad 
        SI edad >= 18 ENTONCES 
            IMPRIMIR "Eres mayor de edad" 
        SINO 
            IMPRIMIR "Eres menor de edad"
        FIN SI 
    FIN

Explicación: Se lee la edad y se evalúa una condición. Dependiendo del resultado, se imprime un mensaje u otro.

Ejemplo 4: Calcular la propina de una comida

Problema: Calcular el 15% de propina a partir del total de una cuenta. 


    INICIO 
        LEER total_cuenta
        propina = total_cuenta * 0.15
        total_a_pagar = total_cuenta + propina 
        IMPRIMIR "Propina: ", propina 
        IMPRIMIR "Total con propina: ", total_a_pagar 
    FIN

Explicación: Se lee el monto de la cuenta, se calcula la propina multiplicando por 0.15, se suma al total original y se muestran ambos resultados.

  1. Sumar dos números ingresados por el usuario.
  2. Calcular el doble de un número.

  6. Calcular el porcentaje de asistencia (dadas clases totales y asistidas).
  7. Intercambiar el valor de dos variables (usando una auxiliar).
  8. Leer la edad y mostrar cuántos meses ha vivido (aprox).
  9. Calcular el costo total de X entradas de cine a precio Y.
