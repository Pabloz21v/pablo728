Sesión 1: Introducción a Bases de Datos (DDBB)

¿Qué es una base de datos?

Una base de datos es una colección organizada de información o datos almacenados de manera estructurada para que sea fácilmente accesible, gestionable y actualizable. Las bases de datos se utilizan para almacenar grandes cantidades de información de manera eficiente y para permitir que los usuarios accedan y manipulen esa información de manera rápida y sencilla.

Tipos de Bases de Datos (DDBB)

  • Bases de Datos Relacionales: Utilizan tablas para almacenar datos y relaciones entre esas tablas. Son las más comunes y utilizan SQL para las consultas.
  • Bases de Datos NoSQL: Diseñadas para manejar datos no estructurados o semiestructurados. Incluyen bases de datos de documentos, clave-valor, columnas anchas y grafos.

Usaremos para bases de datos relacionales MySQL Workbench (interfaz gráfica).

Comandos Básicos en MySQL Workbench

CREATE DATABASE

Para crear una base de datos en MySQL Workbench, escribe el siguiente comando:

CREATE DATABASE nombre_que_le_quiero_dar_a_la_DDBB;

DROP DATABASE

Para eliminar una base de datos, usa el siguiente comando:

DROP DATABASE nombre_que_le_quiero_dar_a_la_DDBB;

Nota: Nunca olvides cerrar la línea de código usando punto y coma (;).

Posicionarnos en una Base de Datos

Usa el comando USE para posicionarte en la base de datos creada:

USE nombre_que_le_quiero_dar_a_la_DDBB;

Para insertar este código, ubica el botón "Create new SQL queries" en la parte superior izquierda, justo debajo del botón "File".

Insertamos en el cuadro de SQL el código anteriormente presentado y damos clic al botón en forma de rayo. Si todo salió bien veremos mensajes en el output (cuadro inferior) indicando que realizó las acciones y mostrando una tilde verde de éxito, sino veremos uno rojo con una cruz dentro.

También podés visualizar dónde están las DDBB que creamos y sus tablas, esto lo podemos hacer ubicando otro botón "Schemas" que se encuentra en la parte inferior del cuadro llamado "Navigator" a su izquierda tenemos otro botón "Administration". Una vez presionado "Schemas" veremos las DDBB instaladas por defecto y las que crearemos.

Creación y Manejo de Tablas

CREATE TABLE

Para crear una tabla, usa el siguiente comando:

CREATE TABLE alumnos ( nombre VARCHAR(25), apellido VARCHAR(25), edad INT, domicilio VARCHAR(80) );

La estructura para crear la tabla necesita de asignarle un nombre, abrir paréntesis () y dentro de él poner el nombre de la columna seguido del tipo de dato que irá en esa columna, cerrando con una coma ,, excepto la última columna, como se puede ver en el ejemplo, y no olvidar terminar la línea de código con el punto y coma ;

En este ejemplo a la columna le pusimos VARCHAR(50), VARCHAR() indica que el tipo de dato que se ingresará en dicha columna será un dato de tipo texto, y el número dentro del paréntesis (50) la cantidad de caracteres permitidos.

Si usamos INT, indicaremos que el tipo de dato a ingresar en esa columna será numérico.

DROP TABLE

Para eliminar una tabla, usa el siguiente comando:

DROP TABLE alumnos;

Si no logramos ver los cambios en la pestaña SCHEMAS, siempre veremos un botón celeste de 2 flechitas formando una ruedita, este botón actualiza la vista de SCHEMAS. Si lo hicimos bien podrán ver las DDBB, tablas y todo cambio actualizado.

Insertar Datos en una Tabla

Ahora vamos a agregar datos a la tabla creada, usa el comando INSERT INTO junto con VALUES:

INSERT INTO alumnos (nombre, apellido, edad, domicilio)
VALUES ('Tony', 'Stark', 1967, 'Avenida siempre viva 123');

Podrán observar que primero usamos INSERT INTO seguido del nombre de la tabla, abrimos paréntesis e ingresamos los nombres de las columnas que queremos llenar con datos (exactamente escritos), posteriormente usamos VALUES, abrimos paréntesis e ingresamos los datos.

Si la columna tenía una configuración VARCHAR, los datos se guardarán como texto, por lo que hay que escribir el nombre rodeado de comillas (apretar el botón de signo de pregunta al lado del 0, generalmente ese botón contiene la comilla indicada). En el ejemplo verán el nombre, apellido y dirección rodeados por dicha comilla.

Cada nombre de tabla y los datos a ingresar siempre deben separarse con coma ,.

No olvides cerrar la línea de código con punto y coma ;.

Respetar el orden ingresado de las columnas con el dato a ingresar, si ingresas un tipo de dato que no corresponde al designado en esa columna, saldrá error y no registrará los cambios.

Ejercicio Práctico

  1. Crear una base de datos con tu nombre.
  2. Crear 3 tablas, cada una con alguna temática de cosas que te gusta (deportistas, videojuegos, moda, autos, aviones, etc).
  3. Asignar nombre a las tablas y el tipo de dato que debería contener: VARCHAR (cantidad de caracteres permitidos) para texto, INT para números.
  4. Agregar por lo menos 3 datos (registros) en cada tabla.
