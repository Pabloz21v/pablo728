🧪 Segmento A: ejemplos de operadores de comparación dentro de if

1. Acceso por edad (Uso de >=)

En este ejemplo comprobamos si un valor es "mayor o igual que" el requerido. Observa cómo, si la condición se cumple (es True), entramos al bloque del if. 

edad = int(input("Ingresa tu edad: "))

if edad >= 18:
    print("Acceso permitido.")
    print("Disfruta de la película.")
else:
    print("Acceso denegado. Eres menor de edad.")

2. Validación de respuesta exacta (Uso de == y !=)

Aquí evaluamos si el texto introducido es idéntico a la respuesta correcta. Si el usuario se equivoca, la condición del if es False y el programa ejecuta el else. 

capital = input("¿Cuál es la capital de Francia?: ")

if capital == "Paris":
    print("¡Correcto! Has ganado 10 puntos.")
else:
    print("Incorrecto. La respuesta era Paris.")

3. Clasificación de temperatura (Uso de <)

Podemos usar el operador "menor que" para activar una advertencia según el valor ingresado. Es muy útil en sistemas que monitorean sensores. 

temperatura = float(input("Ingrese la temperatura del motor: "))

if temperatura < 90.0:
    print("El motor está funcionando a una temperatura segura.")
else:
    print("¡ALERTA! Sobrecalentamiento detectado.")

🧠 Segmento B: ejemplos de operadores lógicos dentro de if

1. Doble validación estricta (Uso de and)

El operador and obliga a que todas las condiciones sean verdaderas. En este simulador de inicio de sesión, tanto el nombre de usuario como la contraseña deben coincidir a la vez. 

usuario = input("Usuario: ")
clave = input("Contraseña: ")

if usuario == "admin" and clave == "secreta123":
    print("Autenticación exitosa. Cargando panel de control...")
else:
    print("Error: Usuario o contraseña incorrectos.")

2. Múltiples caminos válidos (Uso de or)

Con or, basta con que una sola de las condiciones se cumpla. Aquí le damos un descuento al cliente si es mayor de 65 años O si tiene un pase especial. 

edad = int(input("¿Cuántos años tienes?: "))
tiene_pase = input("¿Tienes pase VIP? (si/no): ")

if edad >= 65 or tiene_pase == "si":
    print("¡Felicidades! Tienes entrada gratuita al museo.")
else:
    print("El costo de tu entrada es de $1500.")

3. Invertir la lógica (Uso de not)

El operador not se usa mucho cuando queremos evaluar si algo "no" está sucediendo o cuando tenemos una variable booleana y queremos actuar sobre su estado negativo. 

hay_stock = False

# "not hay_stock" se lee como "si NO hay stock"
if not hay_stock:
    print("Lo sentimos, el producto está agotado.")
else:
    print("Añadiendo producto al carrito de compras.")

🎯 Bloque 1: Prácticas de Comparación Básica

  1. Aprobación escolar: Pide al usuario que ingrese su calificación del 1 al 10 (usa int(input())). Crea un if que verifique si la nota es mayor o igual a 6. Si es así, imprime "Materia Aprobada"; de lo contrario, imprime "Debes rendir examen final".
  2. Contraseña simple: Pide al usuario que configure un PIN numérico. Luego pide que lo repita. Si el primer PIN es distinto de (!=) el segundo, imprime "Los PIN no coinciden". Si son iguales, imprime "PIN guardado con éxito".
  3. Control de peso en ascensor: Pide al usuario ingresar el peso total de la carga (usa float). Si el peso es menor o igual a 300.0 kg, el programa debe mostrar "Ascensor en movimiento". Si lo supera, mostrar "Exceso de peso detectado".

🎯 Bloque 2: Prácticas con Operadores Lógicos

  1. Descuento combinado: Un cine ofrece precio especial si la película es a la mañana ("M") Y si la persona es estudiante ("S"). Pide al usuario el turno y si es estudiante. Usa and para determinar si aplica o no el descuento e imprime un mensaje adecuado.
  2. Selección de idiomas: Pregunta al usuario si habla "Español" o si habla "Inglés". Si al menos uno de los dos idiomas es afirmativo (usa or), imprime "Perfil apto para el puesto de atención al cliente". Si ambos son negativos, imprime "No cumples con el perfil bilingüe".
  3. Sistema sin conexión: Crea una variable booleana hay_conexion = False. Usa el operador not dentro del if para imprimir "Buscando red WiFi..." si no hay conexión, y "Navegando..." en el caso contrario.

✓ Buenas prácticas finales

Al escribir condicionales if combinados, usa paréntesis si tu lógica tiene varios operadores (por ejemplo: if (edad > 18) and (saldo > 100):). Aunque Python muchas veces no los exige obligatoriamente, ayudan muchísimo a que tu código sea legible para ti y para otros programadores. Recuerda siempre mantener tus 4 espacios de indentación intactos al probar estos ejemplos.
