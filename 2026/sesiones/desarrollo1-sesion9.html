Clase 9: Instalación de Python y Primeros Pasos

Marco Teórico

Python es un lenguaje de alto nivel, interpretado y muy legible. Para trabajar necesitamos:

  1. El intérprete de Python (el motor que ejecuta).
  2. Visual Studio Code (VS Code) (el editor donde escribimos - IDE).

Nuestro Primer Comando: print()

La función print() es nuestra forma de pedirle a Python que nos muestre algo en la pantalla. Es como la voz de nuestro programa. El texto debe ir entre comillas.

Regla de oro: Las comillas

Todo el texto que quieras mostrar debe ir entre comillas (" " o ' '). Si no usas comillas, Python pensará que es una instrucción o una variable y se confundirá.

Triples comillas (""" """) al usarlas se respetan los saltos de linea, permitiendo mejorar la estetica de los textos

Guía Rápida

  1. Descargar Python desde python.org e instalar (marcar "Add to PATH").
  2. Descargar VS Code e instalar.
  3. En VS Code, instalar la extensión "Python".
  4. Abrimos el programa y creamos una carpeta
  5. Dentro de la carpeta creamos un archivo /ficheros llamadohola.py
  6. Crear otros archivos / ficheros con la extensión: .php .html .css
  7. Verán que cada icono es diferente, indicando el lenguaje en el que el IDE lo interpretará
  8. Escribir: print("Hola Mundo") y ejecutar (Play).
