Instalación de MySQL Workbench

Descarga

Para instalar MySQL Workbench, sigue estos pasos:

  1. Visita la página oficial de descargas de MySQL Workbench.
  2. Nos aseguramos de elegir el sistema operativo correcto y elegimos el archivo más pesado, que suele ser la versión completa.
  3. Puede aparecer la sugerencia de registrarse o iniciar sesión. Salta esta opción dando clic en el texto inferior "No thanks, just start my download." para descargar sin registrarse.
  4. Una vez descargado, ejecuta el instalador y sigue las instrucciones en pantalla para completar la instalación.

Solución a Errores Comunes

Error 2503 y 2502 en Windows 10 / 11

Si encuentras los errores 2503 o 2502 durante la instalación, sigue estos pasos:

  1. Cierra todas las aplicaciones en segundo plano.
  2. Ejecuta el instalador como administrador.
  3. Desactiva temporalmente el antivirus durante la instalación.
