Clase 10: Variables y Tipos de Datos

Marco Teórico

Una variable es un espacio en memoria con nombre para guardar un dato. En Python se crean asignando un valor con =.

Tipos de datos básicos:

  • int (Enteros): Números sin decimales (10, -5, 2024).
  • float (Flotantes): Números con decimales (3.14, 2.5). Se usa punto, no coma.
  • str (String/Cadena): Texto entre comillas ("Hola", 'Python').
  • bool (Booleanos): Valores de verdad (True, False). Importante la mayúscula inicial.

Tip para el docente

Explique que el nombre de la variable no debe tener espacios ni empezar con números. Use la convención snake_case (ej: nombre_usuario).
