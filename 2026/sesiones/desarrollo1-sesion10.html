Clase 10: 📦 Variables y Tipos de Datos

🔹 1. ¿Qué es una variable?

Una variable es un espacio en la memoria del ordenador que almacena un dato. Podes imaginarla como una caja con una etiqueta (nombre) que guarda un valor (número, texto, etc.). Ese valor puede cambiar durante la ejecución del programa.

Es recomendable nombrar las variables de forma que les sirva de guía sobre que puede estar conteniendo


La variable se encuentra del lado izquierdo acompañada del signo igual "=" el cual se usa pasa indicar que algo se pretende guardar en la variable. Del lado derecho lo que deseamos guardar dentro.


📌 Ejemplos de asignación: 
nombre = "Ana"          #texto (string)
#En esta línea guardamos la palabra Ana (string = cadena de texto") en la variable llamada nombre
          
saludo = "Hola! Mundo informático 2026"     #texto (string)
#En esta línea guardamos en la variable saludo un texto largo

edad = 25               #número entero
#En esta línea se guarda el numero 25 como numero entero en la variable llamada edad (fijarse que no lleva comillas)
#Si guardamos el numero sin comillas la maquina lo interpreta como número, si le agregamos comillas "25" lo interpreta como texto

altura = 1.65           #decimal
#En esta línea guardamos un numero con comas (usamos el punto (.) para determinar que es un numero con coma) y lo guardamos en la variable llamada altura
#Al ser un numero volvemos a escribirlo sin comillas

es_estudiante = True    #booleano
#En esta línea guardamos un booleano (estos pueden ser TRUE = verdadero ó FALSE = falso dentro de la variable es_estudiante
#Para guardar  TRUE o FALSE no debemos rodearlo por comillas, sinó será interpretado como texto y no como booleano

🏷️ 2. Cómo se puede (y no se debe) nombrar una variable

✅ Formas correctas (buenas prácticas)

  • Usar minúsculas y separar con guion bajo: precio_total, nombre_usuario
  • Comenzar con letra o guion bajo: _contador, valor1
  • Solo letras, números y guion bajo (sin espacios ni símbolos especiales ?, -, *) 
mi_variable = 10      #correcto
variable_2 = "Hola"   #correcto
_privado = True       #convención interna

❌ Formas incorrectas

  • No empezar con número: 2edad = 30 ❌
  • No usar espacios: mi variable = 5 ❌
  • No usar palabras reservadas: (propias del lenguaje): if, for, while, print, True, False if = 7, print = 3 ❌
  • Evitar caracteres especiales: mi-variable, mi$variable ❌ 
2edad = 30       #SyntaxError
mi variable = 5  #SyntaxError
if = 7           #SyntaxError

💬 3. Comentarios en Python

Los comentarios son ignorados por Python. Sirven para explicar el código, deshabilitar líneas temporalmente o documentar. 

# Esto es un comentario de una línea

"""
Comentario multilínea:
puedes explicar la lógica completa

"""

Pueden observar 2 cometarios usando # y 1 comentario usando triples doble comillas.

📢 4. Imprimir variables directamente

Podemos pasar solo la variable dentro de print():

precio = 99.90
print(precio)   #vemos en consola: 99.9

#Interpretación: El valor 99.90 guardado dentro de la variable llamada precio es lo que se visualizará una vez lo imprimamos 
#Se visualizará en la consola (pantalla negra en la parte inferior del visual studio code)
precio = 99.90
alimento = "pata de pollo"
vencimiento = "11/04/06"
print(alimento, precio, vencimiento)   #vemos en consola: pata de pollo 99.9 11/04/06

#Acá pueden ver como incorporamos 3 variables en un solo print(), y al imprimirlo vemos los datos de cada variable en el orden que ingresaron dentro el print()
precio = 99.90
alimento = "pata de pollo"
vencimiento = "11/04/06"
print(alimento, precio, precio, precio, alimento, vencimiento, precio, vencimiento)   #vemos en consola: pata de pollo 99.9 99.9 99.9 pata de pollo 11/04/06 99.9 11/04/06

#Acá pueden ver como incorporamos las 3 variables y las usamos tantas veces que queramos en el orden de deseemos
#Pueden observar que se imprimen los valores de cada variable en el orden que se ingresó dentro de print()

🖨️ 5. Incorporar variables y textos en print()

Creamos variables para contener datos dentro con el fin de ser utilizadas posteriormente tantas veces como uno necesite

Ahora verán ejemplos de cómo se usan dichas variables. Y Metodos para usar PRINT()


Dos métodos principales para mostrar el contenido de variables junto con texto:

  • Método 1 (comas): print("Nombre:", nombre, "Edad:", edad) → añade espacios automáticos.
  • Método 2 (f-strings, recomendado): print(f"Nombre: {nombre}, Edad: {edad}") → más legible y potente. 
nombre = "Carlos"
edad = 20

print("El nombre es", nombre, "y tiene", edad, "años")

#Recordemos: lo que está dentro de las comillas se interpreta como texto, la coma fuera de las comillas separa las variables de los textos.
#Interpretación: En esta línea pueden ver fragmentos de texto rodeados por comillas, y por otro lado variables rodeadas por comas (,)
#En el visual studio code podrán diferenciarlos por colores, esto ayuda a que interpreten que es un texto y que se interpreta como variable
#Al imprimirlo podrán observar un texto completo, y verán donde escribieron una variable ahora se encuentra lo que contenía dicha variable.

print(f"El nombre es {nombre} y tiene {edad} años")

#Usando f"" dentro de print() podemos usar las variables al rodearlas con llaves {} y rodear todo por una apertura y cierre de comillas
#Acá usamos otro método para combinar textos y variables el método de f-string, consiste en agregar la letra f y abrimos doble comillas "" contenemos dentro de las comilas tanto textos como variables , pero las variables deben ir rodeadas por llaves {}.
#Cada programador elige el método de comas o f-string que le resulte más práctico, al imprimirlo visualmente serán idénticos.
Copiar el codigo y ejecutarlo. Podrán observar que el resultado es el mismo. Solo cambia la forma de escribir el codigo

📊 6. Tipos de datos básicos y la función type()

Los tipos de datos en Python son clasificaciones que definen el tipo de valor que puede tomar una variable y las operaciones realizables sobre ella. Indican a la computadora cómo manipular la información (números, texto, booleanos) y cómo almacenarla en memoria. Son fundamentales para la estructura y la eficiencia del código

Tipo Qué representa Ejemplo Uso típico
int Números enteros 10, -3, 0, 9999, -805 Edades, cantidades, índices
float Números decimales 3.14, -0.5, 10.15, 1515.99 Alturas, precios, promedios
str Texto (string) "Hola", 'Python', '25', 'Hola Mundo', 'Año 2026' Nombres, mensajes, direcciones
bool Valores lógicos True, False Condiciones, banderas

🔎 Función type()

Nos muestra el tipo de dato de una variable o valor. Muy útil para depurar. 

edad = 25
print(type(edad))        # <class 'int'>

precio = 19.99
print(type(precio))      # <class 'float'>

nombre = "Luis"
print(type(nombre))      # <class 'str'>

es_mayor = False
print(type(es_mayor))    # <class 'bool'>
#Interpretación: dentro del metodo print() agregamos el metodo type() y dentro de type la variable de nuestro interes
#¿Para que hacemos esto? Bueno el interprete "la maquina" utiliza de una forma u otra los valores contenidos dentro de las variables
#por ejemplo queremos hacer una suma pero ingresamos al numero como texto, la maquina no podrá hacer calculos matematicos. (veremos esto con mas profundidad mas adelante)

También puedes pasar directamente el valor:

print(type(3.14))        # <class 'float'class>

#Se ingresa el valor dentro del método type() y al imprimir nos devuelve en la consola como ella "la maquina" lo está interpretando.

📌 Resumen de la clase

  • Variable: caja etiquetada que almacena un dato.
  • Nombres: solo letras, números, guion bajo; no empezar con número; sin espacios.
