Clase 7: Análisis Entrada-Proceso-Salida (EPS)

Marco Teórico

Todo sistema informático funciona bajo el esquema EPS:

  • Entrada (Input): Los datos necesarios que el usuario o el entorno deben proveer. (numeros, textos, imagenes, etc)
  • Proceso: Las operaciones, cálculos o decisiones que se hacen con los datos. (formulas, TRUE/verdadero o FALSE/falso, etc)
  • Salida (Output): El resultado final que se muestra.

Ejemplo 1: Sumar dos números

  • Entrada: Dos números (A y B).
  • Proceso: Sumar A + B.
  • Salida: El resultado de la suma.

Ejemplo 2: Calcular la propina en un restaurante

  • Entrada: El monto total de la cuenta (en pesos o dólares).
  • Proceso: Multiplicar el total de la cuenta por 0.10 (que es el 10%).
  • Salida: La cantidad de dinero que debemos dejar de propina.

Ejemplo 3: Saber si un estudiante aprobó un examen

  • Entrada: La calificación numérica del estudiante (por ejemplo, de 0 a 100).
  • Proceso: Comparar la calificación ingresada con el número 60. Preguntarse: ¿Es mayor o igual a 60?
  • Salida: Un mensaje que diga "Aprobado" o "Reprobado" según el resultado de la comparación.

Ejemplo 4: Calcular la velocidad promedio de un corredor

  • Entrada: La distancia total recorrida (en kilómetros) y el tiempo total que tardó (en horas).
  • Proceso: Dividir la distancia recorrida entre el tiempo empleado (Velocidad = Distancia / Tiempo).
  • Salida: La velocidad promedio (en km/h).

Actividad 7: Definir Entrada, Proceso y Salida

Para cada problema, definir Entrada, Proceso y Salida:

  1. Calcular el área de un rectángulo. (E: base, altura / P: base*altura / S: área).
  2. Calcular el cambio a devolver en una compra.
  3. Convertir metros a centímetros.
  4. Calcular el promedio de 3 notas de un alumno.
  5. Determinar los años que cumple una persona (dado año actual y año nacimiento).
  6. Calcular el precio final de un producto con 21% de IVA.
  7. Dados dos equipos de fútbol y sus goles, decir quién ganó.
  8. Calcular cuántos minutos hay en X horas.
  9. Dividir una cuenta de restaurante entre N amigos.
  10. Calcular el perímetro de un cuadrado.
